## 数学与深度学习的联系

### 核心流程

```
输入数据
    ↓
[线性代数] 矩阵乘法、卷积
    ↓
[初等函数] 激活函数、归一化
    ↓
[概率论] 概率输出
    ↓
[数理统计] 损失函数
    ↓
[微积分] 计算梯度
    ↓
[最优化] 更新参数
    ↓
重复训练...
```

### 常见操作的数学基础

| 操作 | 相关数学 |
|------|----------|
| 全连接层 | 矩阵乘法 |
| 卷积 | 线性代数、信号处理 |
| 激活函数 | 初等函数 |
| Softmax | 指数函数、归一化 |
| 交叉熵损失 | 对数、概率论 |
| 反向传播 | 链式法则、微积分 |
| 批归一化 | 统计量、正态分布 |
| 梯度下降 | 最优化、凸优化 |
| Adam | 动量、指数移动平均 |
| Dropout | 伯努利分布、期望 |
| 学习率调度 | 余弦函数 |

## 学习建议

### 学习顺序

**推荐路径**：
1. **线性代数** → 理解数据表示和矩阵运算
2. **微积分** → 理解梯度和优化
3. **概率论** → 理解损失函数和不确定性
4. **数理统计** → 理解数据分析和评估
5. **最优化方法** → 理解训练算法
6. **初等函数** → 理解激活和损失函数
7. **数列与级数** → 理解学习率调度和序列建模

### 前置知识

- 基础代数运算
- 函数与图像
- 基本编程能力（Python/NumPy）

### 学习目标

完成本教程后，你应该能够：
- ✅ 理解神经网络各组件的数学原理
- ✅ 手动计算简单网络的梯度
- ✅ 选择合适的激活函数和损失函数
- ✅ 理解不同优化器的工作原理
- ✅ 诊断训练问题（梯度消失/爆炸等）

## 数学符号速查表

| 符号 | 含义 | 示例 |
|------|------|------|
| $\in$ | 属于 | $x \in \mathbb{R}$ |
| $\forall$ | 任意 | $\forall x$ |
| $\sum$ | 求和 | $\sum_i x_i$ |
| $\prod$ | 求积 | $\prod_i x_i$ |
| $\int$ | 积分 | $\int f(x)\,dx$ |
| $\partial$ | 偏导 | $\frac{\partial f}{\partial x}$ |
| $\nabla$ | 梯度 | $\nabla f$ |
| $^\top$ | 转置 | $\mathbf{A}^\top$ |
| $^{-1}$ | 逆 | $\mathbf{A}^{-1}$ |
| $\|\mathbf{x}\|$ | 范数 | $\|\mathbf{x}\|_2$ |
| $\mathbb{R}^n$ | n维实数空间 | $\mathbf{x} \in \mathbb{R}^3$ |
| $\mathcal{N}(\mu,\sigma^2)$ | 正态分布 | $X \sim \mathcal{N}(0,1)$ |
| $\mathbb{E}[X]$ | 期望 | $\mathbb{E}[X] = \mu$ |
| $\text{Var}(X)$ | 方差 | $\text{Var}(X) = \sigma^2$ |
