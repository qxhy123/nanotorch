## 概述

YOLO v2 (YOLO9000) 由 Joseph Redmon 和 Ali Farhadi 开发，是 YOLO 系列的第二个版本。它在保持实时性的同时，显著提升了检测精度。

### YOLO v2 的主要特点

1. **Darknet-19 骨干网络**: 19 层卷积网络，比 VGG-16 更快
2. **批归一化 (Batch Normalization)**: 所有卷积层都使用 BN
3. **锚框机制 (Anchor Boxes)**: 使用 K-means 聚类生成锚框
4. **直接位置预测**: 使用 sigmoid 预测相对偏移
5. **Passthrough 层**: 细粒度特征融合
6. **多尺度训练**: 支持不同输入尺寸
7. **高分辨率分类器**: 先在 224×224 训练，再在 448×448 微调

### nanotorch 的 YOLO v2 实现模块

```
nanotorch/detection/yolo_v2/
├── __init__.py        # 模块导出

tests/detection/yolo_v2/

```

## YOLO v2 核心改进

### 1. Darknet-19 骨干网络

Darknet-19 是一个 19 层的卷积网络：

```
Input (3×416×416)
    ↓ Conv 3×3 (32) → MaxPool
    ↓ Conv 3×3 (64) → MaxPool
    ↓ Conv 3×3 (128) → Conv 1×1 (64) → Conv 3×3 (128) → MaxPool
    ↓ Conv 3×3 (256) → Conv 1×1 (128) → Conv 3×3 (256) → MaxPool
    ↓ Conv 3×3 (512) → [Conv 1×1 (256) → Conv 3×3 (512)] ×4 → MaxPool
    ↓ Conv 3×3 (1024) → [Conv 1×1 (512) → Conv 3×3 (1024)] ×2
    ↓ Output (1024×13×13)
```

相比 YOLO v1 的改进：
- **更少参数**: 5.5M vs 40.2M
- **更快速度**: 更高的效率
- **更高精度**: BN 带来的正则化效果

### 2. 批归一化 (Batch Normalization)

YOLO v2 在所有卷积层上使用批归一化：
- **正则化效果**: 减少过拟合
- **加速收敛**: 更快的训练
- **无需 Dropout**: BN 足够

### 3. 锚框机制与 K-means 聚类

YOLO v2 使用 K-means 聚类自动发现最佳锚框：

```python
# VOC 数据集的 5 个锚框
VOC_ANCHORS = [
    (1.3221, 1.73145),   # 小物体
    (3.19275, 4.00944),  # 中等物体
    (5.05587, 8.09892),  # 大物体
    (9.47112, 4.84053),  # 宽物体
    (11.2364, 10.0071)   # 超大物体
]
```

### 4. 直接位置预测

YOLO v2 使用 sigmoid 进行位置预测：

```
bx = σ(tx) + cx
by = σ(ty) + cy
bw = pw * exp(tw)
bh = ph * exp(th)
```

其中：
- `(cx, cy)`: 网格单元左上角坐标
- `(pw, ph)`: 锚框尺寸
- `(tx, ty, tw, th)`: 网络输出

### 5. Passthrough 层

Passthrough 层将高分辨率特征与低分辨率特征融合：

```
高分辨率特征 (26×26×512)
        ↓ Passthrough (stride=2)
        ↓ 重排为 (13×13×2048)
        ↓ Conv 1×1 → (13×13×256)
        ↓ 与主干特征 Concat
        ↓ 最终特征
```

## 模型架构

### 1. ConvBN（基础卷积块）

```python
from nanotorch.detection.yolo_v2 import ConvBN
from nanotorch.tensor import Tensor
import numpy as np

conv_bn = ConvBN(64, 128, kernel_size=3, stride=1, padding=1)
x = Tensor(np.random.randn(1, 64, 52, 52).astype(np.float32))
y = conv_bn(x)
print(y.shape)  # (1, 128, 52, 52)
```

### 2. Darknet19 骨干网络

```python
from nanotorch.detection.yolo_v2 import Darknet19

backbone = Darknet19(in_channels=3)
x = Tensor(np.random.randn(1, 3, 416, 416).astype(np.float32))
features = backbone(x)

print(features['output'].shape)   # (1, 1024, 13, 13)
print(features['route1'].shape)   # (1, 256, 52, 52)
print(features['route2'].shape)   # (1, 512, 26, 26)
```

### 3. Passthrough 层

```python
from nanotorch.detection.yolo_v2 import PassthroughLayer

passthrough = PassthroughLayer(stride=2)
x = Tensor(np.random.randn(1, 512, 26, 26).astype(np.float32))
y = passthrough(x)
print(y.shape)  # (1, 2048, 13, 13)
```

### 4. 完整 YOLOv2 模型

```python
from nanotorch.detection.yolo_v2 import YOLOv2, YOLOv2Tiny, build_yolov2

# 方式一：直接创建
model = YOLOv2(num_classes=20, input_size=416)

# 方式二：创建 Tiny 版本
model = YOLOv2Tiny(num_classes=20, input_size=416)

# 方式三：使用工厂函数
model = build_yolov2('full', num_classes=20, input_size=416)
model = build_yolov2('tiny', num_classes=20, input_size=224)

# 前向传播
x = Tensor(np.random.randn(1, 3, 416, 416).astype(np.float32))
output = model(x)

# 输出形状: (1, 125, 13, 13)
# 125 = 5 anchors × (4 bbox + 1 conf + 20 classes)
print(output['output'].shape)
```

## 损失函数

### YOLO v2 损失函数设计

YOLO v2 使用多部分损失函数：

$$
L = \lambda_{coord} L_{coord} + \lambda_{obj} L_{obj} + \lambda_{noobj} L_{noobj} + \lambda_{class} L_{class}
$$

**坐标损失**（使用 MSE）：

$$
L_{coord} = \sum_{i=0}^{S^2} \sum_{j=0}^{B} \mathbb{1}_{ij}^{obj} \left[ (t_x - \hat{t}_x)^2 + (t_y - \hat{t}_y)^2 + (t_w - \hat{t}_w)^2 + (t_h - \hat{t}_h)^2 \right]
$$

**置信度损失**（使用 BCE）：

$$
L_{obj} = -\sum_{i,j} \mathbb{1}_{ij}^{obj} \left[ C_i \log(\hat{C}_i) + (1 - C_i) \log(1 - \hat{C}_i) \right]
$$

$$
L_{noobj} = -\lambda_{noobj} \sum_{i,j} \mathbb{1}_{ij}^{noobj} \left[ C_i \log(\hat{C}_i) + (1 - C_i) \log(1 - \hat{C}_i) \right]
$$

**分类损失**（使用 Softmax 交叉熵）：

$$
L_{class} = -\sum_{i=0}^{S^2} \mathbb{1}_{i}^{obj} \sum_{c=1}^{C} y_c \log(\hat{p}_c)
$$

### Anchor K-means 聚类

YOLO v2 使用 K-means 聚类自动确定 Anchor 尺寸：

$$
d(\text{box}, \text{centroid}) = 1 - \text{IoU}(\text{box}, \text{centroid})
$$

### Passthrough 层

Passthrough 层将高分辨率特征重新组织：

$$
\text{Output}_{26 \times 26 \times 512} \rightarrow \text{Reshape} \rightarrow \text{Output}_{13 \times 13 \times 2048}
$$

$$
\text{Final} = \text{Concat}(\text{Passthrough}, \text{Conv}_{13 \times 13}) \in \mathbb{R}^{13 \times 13 \times 3072}
$$

### 使用损失函数

```python
from nanotorch.detection.yolo_v2 import YOLOv2Loss

loss_fn = YOLOv2Loss(
    num_classes=20,
    lambda_coord=5.0,     # 坐标损失权重
    lambda_obj=1.0,       # 目标置信度损失权重
    lambda_noobj=0.5,     # 非目标置信度损失权重
    lambda_class=1.0,     # 分类损失权重
    ignore_threshold=0.6  # IoU 忽略阈值
)

# 计算损失
predictions = {
    'output': Tensor(np.random.randn(2, 125, 13, 13).astype(np.float32) * 0.1)
}

targets = [
    {'boxes': np.array([[100, 100, 200, 200]], dtype=np.float32),
     'labels': np.array([0], dtype=np.int64)},
]

loss, loss_dict = loss_fn(predictions, targets)
print(f"Total Loss: {loss.item():.4f}")
print(f"Coord Loss: {loss_dict['coord_loss']:.4f}")
print(f"Obj Loss: {loss_dict['obj_loss']:.4f}")
print(f"NoObj Loss: {loss_dict['noobj_loss']:.4f}")
print(f"Class Loss: {loss_dict['class_loss']:.4f}")
```

## 训练流程

### 完整训练示例

```python
from nanotorch.tensor import Tensor
from nanotorch.optim import Adam
from nanotorch.optim.lr_scheduler import StepLR
from nanotorch.detection.yolo_v2 import build_yolov2, YOLOv2Loss
import numpy as np

# 创建模型
model = build_yolov2('tiny', num_classes=20, input_size=224)

# 损失函数和优化器
loss_fn = YOLOv2Loss(num_classes=20)
optimizer = Adam(model.parameters(), lr=1e-4)
scheduler = StepLR(optimizer, step_size=10, gamma=0.1)

# 训练循环
for epoch in range(10):
    model.train()
    total_loss = 0
    
    for batch_idx in range(10):
        # 生成随机数据
        images = Tensor(np.random.randn(2, 3, 224, 224).astype(np.float32))
        targets = [{
            'boxes': np.random.rand(2, 4).astype(np.float32) * 200,
            'labels': np.random.randint(0, 20, 2).astype(np.int64)
        } for _ in range(2)]
        
        optimizer.zero_grad()
        output = model(images)
        
        # 计算损失
        loss, loss_dict = loss_fn(output, targets)
        
        # 反向传播
        loss.backward()
        optimizer.step()
        
        total_loss += loss.item()
    
    scheduler.step()
    avg_loss = total_loss / 10
    print(f"Epoch {epoch+1}, Loss: {avg_loss:.4f}")
```

## 常见问题

### 1. 为什么使用 K-means 聚类生成锚框？

传统方法使用手工设计的锚框尺寸，而 K-means 聚类：
- 自动发现数据中的最佳锚框尺寸
- 更好地适应不同数据集
- 提高检测精度

### 2. Passthrough 层的作用？

Passthrough 层：
- 将高分辨率特征传递到检测层
- 改善小物体检测
- 类似 ResNet 的跳跃连接

### 3. 直接位置预测的优势？

- **数值稳定**: sigmoid 输出在 [0, 1] 范围
- **物理意义**: 预测相对于网格的偏移
- **收敛更快**: 避免不稳定的位置预测

### 4. Full vs Tiny 版本如何选择？

| 版本 | 参数量 | 速度 | 精度 | 适用场景 |
|------|--------|------|------|----------|
| Full | 5.5M | 中等 | 高 | 服务器/桌面 |
| Tiny | 更少 | 快 | 中等 | 边缘设备 |
