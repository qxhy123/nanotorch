# 第五章（c）：自适应学习率方法

## 自适应学习率的动机

### 固定学习率的问题

在深度网络中，不同参数的梯度特性可能差异很大：
- **频繁更新的参数**：梯度大，应该用较小学习率
- **稀疏更新的参数**：梯度小，应该用较大学习率
- **不同层的参数**：梯度幅度可能差几个数量级

### 自适应的思想

为每个参数 $\theta_i$ 维护独立的学习率 $\eta_i$：

$$
\theta_{i,t+1} = \theta_{i,t} - \eta_{i,t} \cdot g_{i,t}
$$

其中 $\eta_{i,t}$ 根据历史梯度动态调整。

## AdaGrad

### 算法原理

**AdaGrad**（Adaptive Gradient）根据历史梯度平方和调整学习率：

$$
\begin{align}
\mathbf{s}_{t+1} &= \mathbf{s}_t + \mathbf{g}_t \odot \mathbf{g}_t \\
\boldsymbol{\theta}_{t+1} &= \boldsymbol{\theta}_t - \frac{\eta}{\sqrt{\mathbf{s}_{t+1}} + \epsilon} \odot \mathbf{g}_t
\end{align}
$$

其中：
- $\mathbf{s}_t$：累积的梯度平方和
- $\odot$：逐元素乘法
- $\epsilon$：数值稳定性常数（如 $10^{-8}$）

### 直观理解

- **梯度大的参数**：$\sqrt{s}$ 大，有效学习率小
- **梯度小的参数**：$\sqrt{s}$ 小，有效学习率大

### AdaGrad 的实现

```python
import numpy as np

class AdaGrad:
    """AdaGrad 优化器"""
    
    def __init__(self, params, lr=0.01, eps=1e-8):
        self.lr = lr
        self.eps = eps
        self.s = {k: np.zeros_like(v) for k, v in params.items()}
    
    def step(self, params, grads):
        """执行一步更新"""
        for key in params:
            # 累积梯度平方
            self.s[key] += grads[key] ** 2
            
            # 自适应学习率更新
            params[key] -= self.lr * grads[key] / (np.sqrt(self.s[key]) + self.eps)
    
    def get_effective_lr(self, key):
        """获取有效学习率"""
        return self.lr / (np.sqrt(self.s[key]) + self.eps)

# 示例
params = {'w': np.array([1.0, 2.0, 3.0])}
optimizer = AdaGrad(params, lr=0.1)

# 模拟几次更新
for i in range(5):
    grads = {'w': np.array([0.1, 0.5, 0.01])}  # 不同幅度的梯度
    optimizer.step(params, grads)
    print(f"Step {i+1}: params = {params['w']}")
    print(f"         effective_lr = {optimizer.get_effective_lr('w')}")
```

### AdaGrad 的问题

**学习率单调递减**：$\mathbf{s}_t$ 只增不减，导致学习率持续减小，最终可能过早停止学习。

**适用场景**：稀疏数据（如 NLP 中的词嵌入），因为频繁出现的词会获得较小的学习率。

## RMSprop

### 算法原理

**RMSprop**（Root Mean Square Propagation）使用**指数移动平均**替代累积和：

$$
\begin{align}
\mathbf{s}_{t+1} &= \beta \mathbf{s}_t + (1 - \beta) \mathbf{g}_t \odot \mathbf{g}_t \\
\boldsymbol{\theta}_{t+1} &= \boldsymbol{\theta}_t - \frac{\eta}{\sqrt{\mathbf{s}_{t+1}} + \epsilon} \odot \mathbf{g}_t
\end{align}
$$

其中 $\beta$ 是衰减率（通常取 0.9）。

### 与 AdaGrad 的区别

| 特性 | AdaGrad | RMSprop |
|------|---------|---------|
| 梯度平方累积 | 累积和 | 指数移动平均 |
| 学习率变化 | 单调递减 | 动态调整 |
| 长期训练 | 可能停止 | 可以继续 |

### RMSprop 的实现

```python
class RMSprop:
    """RMSprop 优化器"""
    
    def __init__(self, params, lr=0.01, alpha=0.9, eps=1e-8, weight_decay=0):
        self.lr = lr
        self.alpha = alpha  # 衰减率
        self.eps = eps
        self.weight_decay = weight_decay
        self.s = {k: np.zeros_like(v) for k, v in params.items()}
    
    def step(self, params, grads):
        """执行一步更新"""
        for key in params:
            g = grads[key]
            
            # 权重衰减
            if self.weight_decay != 0:
                g = g + self.weight_decay * params[key]
            
            # 更新梯度平方的移动平均
            self.s[key] = self.alpha * self.s[key] + (1 - self.alpha) * (g ** 2)
            
            # RMSprop 更新
            params[key] -= self.lr * g / (np.sqrt(self.s[key]) + self.eps)

# 示例
params = {'w': np.random.randn(10, 5)}
grads = {'w': np.random.randn(10, 5) * 0.1}

optimizer = RMSprop(params, lr=0.01, alpha=0.9)
for _ in range(100):
    optimizer.step(params, grads)
```

## Adam

### 算法原理

**Adam**（Adaptive Moment Estimation）结合了动量和自适应学习率：

$$
\begin{align}
\mathbf{m}_{t+1} &= \beta_1 \mathbf{m}_t + (1 - \beta_1) \mathbf{g}_t & \text{（一阶矩估计）} \\
\mathbf{v}_{t+1} &= \beta_2 \mathbf{v}_t + (1 - \beta_2) \mathbf{g}_t^2 & \text{（二阶矩估计）} \\
\hat{\mathbf{m}}_{t+1} &= \frac{\mathbf{m}_{t+1}}{1 - \beta_1^{t+1}} & \text{（偏差修正）} \\
\hat{\mathbf{v}}_{t+1} &= \frac{\mathbf{v}_{t+1}}{1 - \beta_2^{t+1}} & \text{（偏差修正）} \\
\boldsymbol{\theta}_{t+1} &= \boldsymbol{\theta}_t - \frac{\eta}{\sqrt{\hat{\mathbf{v}}_{t+1}} + \epsilon} \hat{\mathbf{m}}_{t+1}
\end{align}
$$

### 为什么需要偏差修正？

初始化时 $\mathbf{m}_0 = \mathbf{0}$，$\mathbf{v}_0 = \mathbf{0}$，导致初期的估计偏低：

$$
\mathbf{m}_1 = 0.1 \cdot \mathbf{g}_0 \quad \text{（而不是} \approx \mathbf{g}_0 \text{）}
$$

偏差修正通过除以 $(1 - \beta^t)$ 来补偿：

$$
\hat{\mathbf{m}}_1 = \frac{0.1 \cdot \mathbf{g}_0}{1 - 0.9^1} = \mathbf{g}_0
$$

随着 $t \to \infty$，$1 - \beta^t \to 1$，修正效应消失。

**Adam 偏差修正的完整推导**：

**第一步**：展开一阶矩估计 $\mathbf{m}_t$。

$$\mathbf{m}_t = \beta_1 \mathbf{m}_{t-1} + (1-\beta_1)\mathbf{g}_{t-1}$$

递归展开：

$$\mathbf{m}_t = (1-\beta_1)\sum_{i=1}^{t} \beta_1^{t-i} \mathbf{g}_{i-1}$$

**第二步**：计算期望。

假设梯度来自平稳分布，$\mathbb{E}[\mathbf{g}_i] = \mathbf{g}$（真实梯度），则：

$$\mathbb{E}[\mathbf{m}_t] = (1-\beta_1)\mathbf{g}\sum_{i=1}^{t} \beta_1^{t-i} = (1-\beta_1)\mathbf{g}\frac{1-\beta_1^t}{1-\beta_1} = (1-\beta_1^t)\mathbf{g}$$

**第三步**：识别偏差。

$$\mathbb{E}[\mathbf{m}_t] = (1-\beta_1^t)\mathbf{g} \neq \mathbf{g}$$

偏差因子为 $(1-\beta_1^t)$，初期 $t$ 小时偏差大。

**第四步**：构造无偏估计。

$$\hat{\mathbf{m}}_t = \frac{\mathbf{m}_t}{1-\beta_1^t}$$

验证无偏性：

$$\mathbb{E}[\hat{\mathbf{m}}_t] = \frac{\mathbb{E}[\mathbf{m}_t]}{1-\beta_1^t} = \frac{(1-\beta_1^t)\mathbf{g}}{1-\beta_1^t} = \mathbf{g}$$

**第五步**：同理处理二阶矩。

$$\mathbb{E}[\mathbf{v}_t] = (1-\beta_2^t)\mathbb{E}[\mathbf{g}^2]$$

$$\hat{\mathbf{v}}_t = \frac{\mathbf{v}_t}{1-\beta_2^t}$$

**第六步**：收敛性。

当 $t \to \infty$：
- $\beta_1^t \to 0$，$\hat{\mathbf{m}}_t \to \mathbf{m}_t$
- $\beta_2^t \to 0$，$\hat{\mathbf{v}}_t \to \mathbf{v}_t$

$$\boxed{\hat{\mathbf{m}}_t = \frac{\mathbf{m}_t}{1-\beta_1^t}, \quad \hat{\mathbf{v}}_t = \frac{\mathbf{v}_t}{1-\beta_2^t}}$$

**数值示例**（$\beta_1 = 0.9$）：

| $t$ | $1 - \beta_1^t$ | 修正因子 |
|-----|----------------|----------|
| 1 | 0.1 | 10.0 |
| 5 | 0.41 | 2.44 |
| 10 | 0.65 | 1.54 |
| 50 | 0.995 | 1.005 |
| 100 | 0.99997 | 1.00003 |

### Adam 的超参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| $\eta$ | 0.001 | 学习率 |
| $\beta_1$ | 0.9 | 一阶矩衰减率 |
| $\beta_2$ | 0.999 | 二阶矩衰减率 |
| $\epsilon$ | $10^{-8}$ | 数值稳定性 |

**经验**：默认参数通常效果很好，只需调整学习率。

### Adam 的实现

```python
class Adam:
    """Adam 优化器"""
    
    def __init__(self, params, lr=0.001, betas=(0.9, 0.999), eps=1e-8, weight_decay=0):
        self.lr = lr
        self.beta1, self.beta2 = betas
        self.eps = eps
        self.weight_decay = weight_decay
        self.t = 0
        
        # 初始化状态
        self.m = {k: np.zeros_like(v) for k, v in params.items()}  # 一阶矩
        self.v = {k: np.zeros_like(v) for k, v in params.items()}  # 二阶矩
    
    def step(self, params, grads):
        """执行一步更新"""
        self.t += 1
        
        for key in params:
            g = grads[key]
            
            # 权重衰减（L2 正则化）
            if self.weight_decay != 0:
                g = g + self.weight_decay * params[key]
            
            # 更新一阶矩
            self.m[key] = self.beta1 * self.m[key] + (1 - self.beta1) * g
            
            # 更新二阶矩
            self.v[key] = self.beta2 * self.v[key] + (1 - self.beta2) * (g ** 2)
            
            # 偏差修正
            m_hat = self.m[key] / (1 - self.beta1 ** self.t)
            v_hat = self.v[key] / (1 - self.beta2 ** self.t)
            
            # Adam 更新
            params[key] -= self.lr * m_hat / (np.sqrt(v_hat) + self.eps)
    
    def get_lr(self):
        return self.lr

# 使用示例
params = {
    'w1': np.random.randn(784, 256) * 0.01,
    'b1': np.zeros(256),
    'w2': np.random.randn(256, 10) * 0.01,
    'b2': np.zeros(10)
}

optimizer = Adam(params, lr=0.001, betas=(0.9, 0.999))

# 模拟训练
for epoch in range(10):
    grads = {k: np.random.randn(*v.shape) * 0.01 for k, v in params.items()}
    optimizer.step(params, grads)
    print(f"Epoch {epoch+1}, w1 norm: {np.linalg.norm(params['w1']):.4f}")
```

### Adam 的优势

1. **自适应学习率**：每个参数有独立的学习率
2. **动量效果**：利用历史梯度信息
3. **偏差修正**：初期更稳定
4. **超参数鲁棒**：默认参数通常有效
5. **内存效率**：只需存储 $\mathbf{m}$ 和 $\mathbf{v}$

## AdamW

### 问题：Adam 中的权重衰减

在 Adam 中，权重衰减通常通过 $L_2$ 正则化实现：

$$
\mathbf{g} = \nabla f(\boldsymbol{\theta}) + \lambda \boldsymbol{\theta}
$$

但这与"真正的"权重衰减不同，因为梯度被自适应地缩放了。

### AdamW 的解决方案

**AdamW**（Adam with Decoupled Weight Decay）将权重衰减与梯度更新解耦：

$$
\boldsymbol{\theta}_{t+1} = \boldsymbol{\theta}_t - \eta \left( \frac{\hat{\mathbf{m}}_{t+1}}{\sqrt{\hat{\mathbf{v}}_{t+1}} + \epsilon} + \lambda \boldsymbol{\theta}_t \right)
$$

权重衰减直接应用于参数，而不是加入梯度。

### AdamW 的实现

```python
class AdamW:
    """AdamW 优化器（解耦权重衰减）"""
    
    def __init__(self, params, lr=0.001, betas=(0.9, 0.999), eps=1e-8, weight_decay=0.01):
        self.lr = lr
        self.beta1, self.beta2 = betas
        self.eps = eps
        self.weight_decay = weight_decay
        self.t = 0
        
        self.m = {k: np.zeros_like(v) for k, v in params.items()}
        self.v = {k: np.zeros_like(v) for k, v in params.items()}
    
    def step(self, params, grads):
        """执行一步更新"""
        self.t += 1
        
        for key in params:
            g = grads[key]
            
            # 更新一阶矩和二阶矩（不包含权重衰减）
            self.m[key] = self.beta1 * self.m[key] + (1 - self.beta1) * g
            self.v[key] = self.beta2 * self.v[key] + (1 - self.beta2) * (g ** 2)
            
            # 偏差修正
            m_hat = self.m[key] / (1 - self.beta1 ** self.t)
            v_hat = self.v[key] / (1 - self.beta2 ** self.t)
            
            # AdamW 更新：权重衰减与梯度更新分离
            params[key] -= self.lr * (
                m_hat / (np.sqrt(v_hat) + self.eps) + 
                self.weight_decay * params[key]  # 解耦的权重衰减
            )

# 比较 Adam 和 AdamW
def compare_adam_adamw():
    """比较 Adam 和 AdamW 的行为"""
    np.random.seed(42)
    
    # 初始参数
    params_adam = {'w': np.random.randn(100) * 0.1}
    params_adamw = {'w': params_adam['w'].copy()}
    
    optimizer_adam = Adam(params_adam, lr=0.01, weight_decay=0.01)
    optimizer_adamw = AdamW(params_adamw, lr=0.01, weight_decay=0.01)
    
    # 记录权重范数
    norm_adam = [np.linalg.norm(params_adam['w'])]
    norm_adamw = [np.linalg.norm(params_adamw['w'])]
    
    for _ in range(100):
        grads = {'w': np.random.randn(100) * 0.1}
        
        optimizer_adam.step(params_adam, grads)
        optimizer_adamw.step(params_adamw, grads)
        
        norm_adam.append(np.linalg.norm(params_adam['w']))
        norm_adamw.append(np.linalg.norm(params_adamw['w']))
    
    import matplotlib.pyplot as plt
    plt.figure(figsize=(10, 6))
    plt.plot(norm_adam, label='Adam (L2 正则)', linewidth=2)
    plt.plot(norm_adamw, label='AdamW (解耦权重衰减)', linewidth=2)
    plt.xlabel('迭代次数')
    plt.ylabel('权重范数')
    plt.title('Adam vs AdamW: 权重衰减效果比较')
    plt.legend()
    plt.grid(True)
    plt.show()

compare_adam_adamw()
```

### AdamW 的优势

1. **更好的泛化**：真正的权重衰减效果
2. **与学习率调度配合更好**：权重衰减不受自适应缩放影响
3. **Transformer 训练的标准选择**

## 优化器比较与选择

### 收敛性比较

| 优化器 | 类型 | 内存 | 适用场景 |

### 实验比较

```python
def compare_optimizers():
    """比较不同优化器在同一问题上的表现"""
    import matplotlib.pyplot as plt
    
    # 定义目标函数（非凸）
    def f(x):
        return x[0]**2 + 10*x[1]**2 + 0.1*np.sin(5*x[0])*np.cos(5*x[1])
    
    def grad_f(x):
        return np.array([
            2*x[0] + 0.5*np.cos(5*x[0])*np.cos(5*x[1]),
            20*x[1] - 0.5*np.sin(5*x[0])*np.sin(5*x[1])
        ])
    
    x0 = np.array([2.0, 2.0])
    n_iters = 200
    
    # SGD with Momentum
    params_sgd = {'x': x0.copy()}
    opt_sgd = SGDMomentum(params_sgd, lr=0.01, momentum=0.9)
    f_sgd = [f(params_sgd['x'])]
    for _ in range(n_iters):
        g = {'x': grad_f(params_sgd['x'])}
        opt_sgd.step(params_sgd, g)
        f_sgd.append(f(params_sgd['x']))
    
    # Adam
    params_adam = {'x': x0.copy()}
    opt_adam = Adam(params_adam, lr=0.1)
    f_adam = [f(params_adam['x'])]
    for _ in range(n_iters):
        g = {'x': grad_f(params_adam['x'])}
        opt_adam.step(params_adam, g)
        f_adam.append(f(params_adam['x']))
    
    # RMSprop
    params_rms = {'x': x0.copy()}
    opt_rms = RMSprop(params_rms, lr=0.1)
    f_rms = [f(params_rms['x'])]
    for _ in range(n_iters):
        g = {'x': grad_f(params_rms['x'])}
        opt_rms.step(params_rms, g)
        f_rms.append(f(params_rms['x']))
    
    plt.figure(figsize=(12, 5))
    
    plt.subplot(1, 2, 1)
    plt.plot(f_sgd, label='SGD + Momentum', linewidth=2)
    plt.plot(f_adam, label='Adam', linewidth=2)
    plt.plot(f_rms, label='RMSprop', linewidth=2)
    plt.xlabel('迭代次数')
    plt.ylabel('函数值')
    plt.title('收敛曲线')
    plt.legend()
    plt.grid(True)
    
    plt.subplot(1, 2, 2)
    plt.semilogy(f_sgd, label='SGD + Momentum', linewidth=2)
    plt.semilogy(f_adam, label='Adam', linewidth=2)
    plt.semilogy(f_rms, label='RMSprop', linewidth=2)
    plt.xlabel('迭代次数')
    plt.ylabel('函数值（对数）')
    plt.title('收敛曲线（对数坐标）')
    plt.legend()
    plt.grid(True)
    
    plt.tight_layout()
    plt.show()

# 辅助类
class SGDMomentum:
    def __init__(self, params, lr, momentum):
        self.lr = lr
        self.momentum = momentum
        self.v = {k: np.zeros_like(v) for k, v in params.items()}
    
    def step(self, params, grads):
        for key in params:
            self.v[key] = self.momentum * self.v[key] + grads[key]
            params[key] -= self.lr * self.v[key]

compare_optimizers()
```

## 在深度学习中的应用

### nanotorch 中的优化器

```python
from nanotorch.optim import SGD, Adam, AdamW, RMSprop
from nanotorch import Tensor
from nanotorch.nn import Linear, ReLU, Sequential, CrossEntropyLoss

# 创建模型
model = Sequential(
    Linear(784, 256),
    ReLU(),
    Linear(256, 128),
    ReLU(),
    Linear(128, 10)
)

# 选择优化器
# 选项 1: SGD + Momentum（适合 CV）
optimizer = SGD(model.parameters(), lr=0.1, momentum=0.9)

# 选项 2: Adam（快速原型）
optimizer = Adam(model.parameters(), lr=0.001)

# 选项 3: AdamW（适合 Transformer）
optimizer = AdamW(model.parameters(), lr=0.001, weight_decay=0.01)

# 训练循环
criterion = CrossEntropyLoss()

for epoch in range(100):
    for batch_x, batch_y in train_loader:
        # 前向传播
        output = model(Tensor(batch_x))
        loss = criterion(output, Tensor(batch_y))
        
        # 反向传播
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
    
    if epoch % 10 == 0:
        print(f"Epoch {epoch}, Loss: {loss.item():.4f}")
```

### 梯度裁剪

对于 RNN 或某些不稳定的情况，梯度裁剪很重要：

```python
def clip_grad_norm_(grads, max_norm):
    """梯度范数裁剪"""
    total_norm = np.sqrt(sum(np.sum(g ** 2) for g in grads.values()))
    if total_norm > max_norm:
        scale = max_norm / total_norm
        for key in grads:
            grads[key] *= scale
    return total_norm

# 在训练中使用
for batch_x, batch_y in train_loader:
    output = model(Tensor(batch_x))
    loss = criterion(output, Tensor(batch_y))
    
    optimizer.zero_grad()
    loss.backward()
    
    # 梯度裁剪
    clip_grad_norm_(model.grads(), max_norm=1.0)
    
    optimizer.step()
```

## 小结

本节介绍了自适应学习率方法：

| 方法 | 核心思想 | 优点 | 缺点 |
|------|----------|------|------|
| AdaGrad | 累积梯度平方 | 稀疏数据友好 | 学习率递减 |
| RMSprop | EMA 梯度平方 | 非平稳目标 | 需调参 |
| Adam | 动量 + 自适应 | 通用、鲁棒 | 可能过拟合 |
| AdamW | Adam + 解耦正则 | 更好泛化 | - |

**关键要点**：
- Adam 是最常用的优化器，默认参数通常有效
- AdamW 在 Transformer 训练中表现更好
- SGD + Momentum 在精细调优时可能获得更好的泛化
- 自适应方法简化了学习率调优
