## 最优化问题概述

### 一般形式

最优化问题的标准形式为：

$$
\min_{\mathbf{x} \in \mathbb{R}^n} f(\mathbf{x})
$$

其中：
- $f: \mathbb{R}^n \to \mathbb{R}$ 是**目标函数**（损失函数）
- $\mathbf{x} = (x_1, x_2, \ldots, x_n)^\top$ 是**决策变量**（模型参数）

### 约束优化问题

带有约束的最优化问题：

$$
\begin{align}
\min_{\mathbf{x}} \quad & f(\mathbf{x}) \\
\text{s.t.} \quad & g_i(\mathbf{x}) \leq 0, \quad i = 1, \ldots, m \quad \text{（不等式约束）} \\
& h_j(\mathbf{x}) = 0, \quad j = 1, \ldots, p \quad \text{（等式约束）}
\end{align}
$$

### 最优性条件

**全局最优**：$\mathbf{x}^*$ 满足 $f(\mathbf{x}^*) \leq f(\mathbf{x}), \forall \mathbf{x} \in \mathbb{R}^n$

**局部最优**：$\mathbf{x}^*$ 满足存在 $\epsilon > 0$，使得 $f(\mathbf{x}^*) \leq f(\mathbf{x}), \forall \mathbf{x} \in B_\epsilon(\mathbf{x}^*)$

**一阶必要条件**（无约束）：

若 $\mathbf{x}^*$ 是局部最优且 $f$ 在 $\mathbf{x}^*$ 处可微，则：

$$
\nabla f(\mathbf{x}^*) = \mathbf{0}
$$

**二阶必要条件**：

若 $\mathbf{x}^*$ 是局部最优且 $f$ 二阶可微，则：

$$
\nabla^2 f(\mathbf{x}^*) \succeq 0 \quad \text{（Hessian 矩阵半正定）}
$$

**二阶充分条件**：

若 $\nabla f(\mathbf{x}^*) = \mathbf{0}$ 且 $\nabla^2 f(\mathbf{x}^*) \succ 0$（正定），则 $\mathbf{x}^*$ 是严格局部最优。

```python
import numpy as np

def check_optimality_conditions(f, grad_f, hess_f, x):
    """检查最优性条件"""
    grad = grad_f(x)
    hess = hess_f(x)
    
    # 一阶条件：梯度是否接近零
    grad_norm = np.linalg.norm(grad)
    first_order = grad_norm < 1e-6
    
    # 二阶条件：Hessian 是否半正定
    eigenvalues = np.linalg.eigvalsh(hess)
    second_order = np.all(eigenvalues >= -1e-10)
    
    return {
        'gradient_norm': grad_norm,
        'first_order_satisfied': first_order,
        'hessian_eigenvalues': eigenvalues,
        'second_order_satisfied': second_order
    }

# 示例：f(x) = x^2
f = lambda x: x**2
grad_f = lambda x: 2*x
hess_f = lambda x: np.array([[2.0]])

result = check_optimality_conditions(f, grad_f, hess_f, np.array([0.0]))
print(f"最优点检查: {result}")
```

## 凸优化基础

### 凸集

**定义**：集合 $C \subseteq \mathbb{R}^n$ 是凸集，当且仅当：

$$
\forall \mathbf{x}, \mathbf{y} \in C, \forall \theta \in [0, 1]: \quad \theta \mathbf{x} + (1 - \theta) \mathbf{y} \in C
$$

**直观理解**：集合中任意两点的连线仍在集合内。

**常见凸集**：
- **超平面**：$\{\mathbf{x} : \mathbf{a}^\top \mathbf{x} = b\}$
- **半空间**：$\{\mathbf{x} : \mathbf{a}^\top \mathbf{x} \leq b\}$
- **球**：$\{\mathbf{x} : \|\mathbf{x} - \mathbf{x}_0\|_2 \leq r\}$
- **范数球**：$\{\mathbf{x} : \|\mathbf{x}\|_p \leq 1\}$
- **多面体**：有限个半空间的交集

### 凸函数

**定义**：函数 $f: \mathbb{R}^n \to \mathbb{R}$ 是凸函数，当且仅当：

$$
f(\theta \mathbf{x} + (1 - \theta) \mathbf{y}) \leq \theta f(\mathbf{x}) + (1 - \theta) f(\mathbf{y}), \quad \forall \theta \in [0, 1]
$$

**一阶条件**（Jensen 不等式）：$f$ 可微时，$f$ 是凸函数当且仅当：

$$
f(\mathbf{y}) \geq f(\mathbf{x}) + \nabla f(\mathbf{x})^\top (\mathbf{y} - \mathbf{x}), \quad \forall \mathbf{x}, \mathbf{y}
$$

**二阶条件**：$f$ 二阶可微时，$f$ 是凸函数当且仅当：

$$
\nabla^2 f(\mathbf{x}) \succeq 0, \quad \forall \mathbf{x}
$$

### 强凸性

函数 $f$ 是 $\mu$-强凸的，如果：

$$
f(\mathbf{y}) \geq f(\mathbf{x}) + \nabla f(\mathbf{x})^\top (\mathbf{y} - \mathbf{x}) + \frac{\mu}{2}\|\mathbf{y} - \mathbf{x}\|^2
$$

**等价条件**：$\nabla^2 f(\mathbf{x}) \succeq \mu \mathbf{I}$

**意义**：强凸性保证目标函数有"碗状"形状，存在唯一全局最优。

### 光滑性

函数 $f$ 是 $L$-光滑的（Lipschitz 连续梯度），如果：

$$
\|\nabla f(\mathbf{x}) - \nabla f(\mathbf{y})\| \leq L \|\mathbf{x} - \mathbf{y}\|
$$

**等价条件**：$\nabla^2 f(\mathbf{x}) \preceq L \mathbf{I}$

### 凸优化的重要性

**关键性质**：对于凸优化问题，**局部最优 = 全局最优**

这意味着：
1. 驻点（$\nabla f(\mathbf{x}) = 0$）必定是全局最优
2. 任何局部搜索算法都能找到全局最优
3. 理论分析更加简洁

```python
import numpy as np
```

## 梯度下降法

### 基本算法

**梯度下降**是最基础的一阶优化方法：

$$
\mathbf{x}_{t+1} = \mathbf{x}_t - \eta \nabla f(\mathbf{x}_t)
$$

其中 $\eta > 0$ 是**学习率**（步长）。

### 直观理解

梯度 $\nabla f(\mathbf{x})$ 指向函数值**增长最快**的方向，因此沿着负梯度方向移动可以降低函数值。

**泰勒展开视角**：

$$
f(\mathbf{x}_{t+1}) \approx f(\mathbf{x}_t) + \nabla f(\mathbf{x}_t)^\top (\mathbf{x}_{t+1} - \mathbf{x}_t) = f(\mathbf{x}_t) - \eta \|\nabla f(\mathbf{x}_t)\|^2
$$

只要 $\eta > 0$ 且梯度非零，函数值就会下降。

### 梯度下降的几何解释

```
等高线图上的轨迹：

    x₂
     ↑
     │    ∘ 初始点
     │   ∘
     │  ∘
     │ ∘
     │∘ → 沿负梯度方向
     └──────────→ x₁
     
目标：到达最低点（等高线中心）
```

### 算法实现

```python
import numpy as np

def gradient_descent(f, grad_f, x0, learning_rate=0.01, max_iters=1000, tol=1e-6):
    """
    梯度下降算法
    
    Args:
        f: 目标函数
        grad_f: 梯度函数
        x0: 初始点
        learning_rate: 学习率
        max_iters: 最大迭代次数
        tol: 收敛容差
    
    Returns:
        x: 最优解
        history: 迭代历史
    """
    x = x0.copy()
    history = {'x': [x.copy()], 'f': [f(x)], 'grad_norm': []}
    
    for i in range(max_iters):
        g = grad_f(x)
        grad_norm = np.linalg.norm(g)
        history['grad_norm'].append(grad_norm)
        
        # 检查收敛
        if grad_norm < tol:
            print(f"在 {i} 次迭代后收敛")
            break
        
        # 梯度下降更新
        x = x - learning_rate * g
        
        history['x'].append(x.copy())
        history['f'].append(f(x))
    
    return x, history

# 示例：最小化 f(x) = x₁² + 2x₂²
def f(x):
    return x[0]**2 + 2 * x[1]**2

def grad_f(x):
    return np.array([2 * x[0], 4 * x[1]])

x0 = np.array([2.0, 2.0])
x_opt, history = gradient_descent(f, grad_f, x0, learning_rate=0.1)
print(f"最优解: {x_opt}")
print(f"迭代次数: {len(history['f'])}")
print(f"最优值: {history['f'][-1]:.6f}")
```

## 步长选择策略

### 步长的重要性

学习率 $\eta$ 是梯度下降中最关键的超参数：
- **太大**：可能发散或震荡
- **太小**：收敛太慢
- **刚好**：快速稳定收敛

### 固定步长

$$
\eta = \text{const}
$$

**理论最优值**：对于 $L$-光滑函数，最优固定步长为 $\eta = \frac{1}{L}$。

### 精确线搜索

$$
\eta_t = \arg\min_\eta f(\mathbf{x}_t - \eta \nabla f(\mathbf{x}_t))
$$

**优点**：每步最优  
**缺点**：计算代价高，通常不可行

### 回溯线搜索 (Backtracking Line Search)

**Armijo 条件**（充分下降）：

$$
f(\mathbf{x}_t - \eta \nabla f(\mathbf{x}_t)) \leq f(\mathbf{x}_t) - c \eta \|\nabla f(\mathbf{x}_t)\|^2, \quad c \in (0, 1)
$$

**算法**：
1. 初始化 $\eta = \eta_0$（如 1.0）
2. 如果不满足 Armijo 条件，$\eta := \beta \eta$（如 $\beta = 0.5$）
3. 重复直到满足条件

```python
def backtracking_line_search(f, grad_f, x, d, c=0.5, beta=0.8, max_iters=20):
    """
    回溯线搜索
    
    Args:
        f: 目标函数
        grad_f: 梯度函数
        x: 当前点
        d: 搜索方向（通常是负梯度）
        c: Armijo 参数
        beta: 步长衰减率
        max_iters: 最大回溯次数
    """
    eta = 1.0
    g = grad_f(x)
    f_x = f(x)
    
    for _ in range(max_iters):
        x_new = x + eta * d
        if f(x_new) <= f_x + c * eta * np.dot(g, d):
            return eta
        eta *= beta
    
    return eta

def gradient_descent_backtracking(f, grad_f, x0, max_iters=1000, tol=1e-6):
    """带回溯线搜索的梯度下降"""
    x = x0.copy()
    history = {'x': [x.copy()], 'f': [f(x)], 'lr': []}
    
    for i in range(max_iters):
        g = grad_f(x)
        
        if np.linalg.norm(g) < tol:
            break
        
        # 回溯线搜索确定步长
        eta = backtracking_line_search(f, grad_f, x, -g)
        history['lr'].append(eta)
        
        x = x - eta * g
        history['x'].append(x.copy())
        history['f'].append(f(x))
    
    return x, history
```

### 递减步长

对于非凸问题或随机优化：

$$
\eta_t = \frac{\eta_0}{1 + \alpha t}
$$

或

$$
\eta_t = \frac{\eta_0}{\sqrt{t}}
$$

### 步长选择总结

| 策略 | 公式 | 特点 |
|------|------|------|
| 固定步长 | $\eta = \frac{1}{L}$ | 需要知道光滑常数 $L$ |
| 精确线搜索 | $\eta_t = \arg\min_\eta f(x_t - \eta g_t)$ | 计算代价高 |
| 回溯线搜索 | 逐步减小直到满足 Armijo | 实用性强 |
| 递减步长 | $\eta_t = \eta_0/(1+\alpha t)$ | 适用于非凸问题 |

## 收敛性分析

### 收敛率定义

设 $\{\mathbf{x}_t\}$ 是优化算法产生的序列，$\mathbf{x}^*$ 是最优解。

| 类型 | 定义 | 含义 |
|------|------|------|
| 次线性 | $\|\mathbf{x}_t - \mathbf{x}^*\| = O(1/t)$ | 精度提高一阶需要固定次数迭代 |
| 线性 | $\|\mathbf{x}_t - \mathbf{x}^*\| = O(r^t), r < 1$ | 每迭代一次精度乘以常数 |
| 超线性 | $\|\mathbf{x}_{t+1} - \mathbf{x}^*\| / \|\mathbf{x}_t - \mathbf{x}^*\| \to 0$ | 收敛越来越快 |
| 二次 | $\|\mathbf{x}_{t+1} - \mathbf{x}^*\| = O(\|\mathbf{x}_t - \mathbf{x}^*\|^2)$ | 精度位数翻倍 |

### 凸函数的收敛性

**定理**（凸 + $L$-光滑）：设 $f$ 是凸函数且 $L$-光滑，使用固定步长 $\eta \leq \frac{1}{L}$，则：

$$
f(\mathbf{x}_T) - f(\mathbf{x}^*) \leq \frac{\|\mathbf{x}_0 - \mathbf{x}^*\|^2}{2 \eta T}
$$

**凸函数梯度下降收敛性证明**：

**第一步**：利用 $L$-光滑性（Lipschitz 连续梯度）。

如果 $f$ 是 $L$-光滑的，则对于任意 $\mathbf{x}, \mathbf{y}$：

$$f(\mathbf{y}) \leq f(\mathbf{x}) + \nabla f(\mathbf{x})^\top(\mathbf{y} - \mathbf{x}) + \frac{L}{2}\|\mathbf{y} - \mathbf{x}\|^2$$

**第二步**：设 $\mathbf{y} = \mathbf{x}_{t+1} = \mathbf{x}_t - \eta \nabla f(\mathbf{x}_t)$。

$$f(\mathbf{x}_{t+1}) \leq f(\mathbf{x}_t) + \nabla f(\mathbf{x}_t)^\top(-\eta \nabla f(\mathbf{x}_t)) + \frac{L}{2}\|\eta \nabla f(\mathbf{x}_t)\|^2$$

$$= f(\mathbf{x}_t) - \eta \|\nabla f(\mathbf{x}_t)\|^2 + \frac{L\eta^2}{2}\|\nabla f(\mathbf{x}_t)\|^2$$

$$= f(\mathbf{x}_t) - \eta\left(1 - \frac{L\eta}{2}\right)\|\nabla f(\mathbf{x}_t)\|^2$$

**第三步**：当 $\eta \leq \frac{1}{L}$ 时，$1 - \frac{L\eta}{2} \geq \frac{1}{2}$，因此：

$$f(\mathbf{x}_{t+1}) \leq f(\mathbf{x}_t) - \frac{\eta}{2}\|\nabla f(\mathbf{x}_t)\|^2$$

**第四步**：利用凸性，$f(\mathbf{x}^*) \geq f(\mathbf{x}_t) + \nabla f(\mathbf{x}_t)^\top(\mathbf{x}^* - \mathbf{x}_t)$。

移项得：$\|\nabla f(\mathbf{x}_t)\| \cdot \|\mathbf{x}_t - \mathbf{x}^*\| \geq f(\mathbf{x}_t) - f(\mathbf{x}^*)$

**第五步**：结合以上结果，可以证明：

$$f(\mathbf{x}_T) - f(\mathbf{x}^*) \leq \frac{\|\mathbf{x}_0 - \mathbf{x}^*\|^2}{2\eta T}$$

**结论**：$O(1/T)$ 收敛率（次线性）。

### 强凸函数的收敛性

**定理**（$\mu$-强凸 + $L$-光滑）：设 $f$ 是 $\mu$-强凸且 $L$-光滑，条件数 $\kappa = L/\mu$，使用最优步长 $\eta = 1/L$，则：

$$
\|\mathbf{x}_T - \mathbf{x}^*\|^2 \leq \left(1 - \frac{1}{\kappa}\right)^T \|\mathbf{x}_0 - \mathbf{x}^*\|^2
$$

**结论**：线性收敛率，收敛速度取决于条件数 $\kappa$。

### 条件数的影响

**条件数** $\kappa = L/\mu$ 反映了问题的"难度"：
- $\kappa$ 小（接近 1）：目标函数接近圆形等高线，易优化
- $\kappa$ 大：目标函数狭长（峡谷），难优化

```
条件数小 (κ ≈ 1)          条件数大 (κ >> 1)
    ○                       ═══
  ○   ○                   ═══════
 ○     ○                ═══════════
○       ○              ═══════════════
 ○     ○                ═══════════
  ○   ○                   ═══════
    ○                       ═══
    
  快速收敛                  震荡收敛
```

## 在深度学习中的应用

### 随机梯度下降 (SGD)

在深度学习中，目标函数通常是训练样本损失的平均：

$$
f(\mathbf{w}) = \frac{1}{N} \sum_{i=1}^N \ell(\mathbf{w}; \mathbf{x}_i, y_i)
$$

**随机梯度**：使用单个样本或 mini-batch 估计梯度：

$$
\mathbf{g}_t = \frac{1}{B} \sum_{i \in \mathcal{B}_t} \nabla \ell(\mathbf{w}_t; \mathbf{x}_i, y_i) \approx \nabla f(\mathbf{w}_t)
$$

```python
def sgd_step(params, grads, lr):
    """SGD 参数更新"""
    for key in params:
        params[key] -= lr * grads[key]
    return params

def mini_batch_sgd(model, data_loader, lr, epochs):
    """Mini-batch SGD 训练循环"""
    for epoch in range(epochs):
        for batch_x, batch_y in data_loader:
            # 前向传播
            output = model.forward(batch_x)
            
            # 计算损失和梯度
            loss = model.compute_loss(output, batch_y)
            grads = model.backward()
            
            # SGD 更新
            model.params = sgd_step(model.params, grads, lr)
```

### 深度学习中的非凸性

神经网络的损失函数是**非凸**的：
- 存在大量局部最优
- 存在鞍点
- 局部最优 ≠ 全局最优

**实践中的发现**：
- 大多数局部最优质量相近
- 鞍点比局部最优更常见
- SGD 的噪声有助于逃离鞍点

### 实践建议

1. **学习率选择**
   - 从小值开始（如 0.001）
   - 使用学习率查找器
   - 监控训练曲线

2. **Batch Size**
   - 小 batch：正则化效果强，噪声大
   - 大 batch：收敛快，泛化可能差

3. **初始化**
   - 使用适当的权重初始化（Xavier, He）
   - 避免全零初始化

## 小结

本节介绍了优化基础和梯度下降的核心概念：

| 概念 | 定义/公式 | 重要性 |
|------|----------|--------|
| 凸函数 | $f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y)$ | 局部最优 = 全局最优 |
| 强凸性 | $\nabla^2 f \succeq \mu I$ | 保证线性收敛 |
| 光滑性 | $\nabla^2 f \preceq L I$ | 确定最大步长 |
| 梯度下降 | $x_{t+1} = x_t - \eta \nabla f(x_t)$ | 最基础的一阶方法 |
| 条件数 | $\kappa = L/\mu$ | 决定收敛难度 |

**关键要点**：
- 凸优化有理论保证，但深度学习是非凸的
- 学习率是最重要的超参数
- 条件数影响收敛速度
- 随机梯度下降是深度学习的标准方法
