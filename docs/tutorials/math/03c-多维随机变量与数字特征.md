## 多维随机变量概述

### 定义

**多维随机变量**（随机向量）是由多个随机变量组成的向量：

$$
\mathbf{X} = (X_1, X_2, \ldots, X_n)^\top
$$

### 示例

| 场景 | 随机变量 |
|------|----------|
| 图像分类 | 像素值 $(X_1, X_2, \ldots, X_{784})$ |
| 自然语言 | 词向量 $(X_1, X_2, \ldots, X_d)$ |
| 时间序列 | 时刻 t 的状态 $(X_t, Y_t, Z_t)$ |
| 多标签分类 | 各标签的预测概率 $(p_1, p_2, \ldots, p_K)$ |

### 为什么重要？

深度学习本质上是在处理**高维随机变量**：
- **输入层**：每个特征是一个随机变量
- **隐藏层**：激活值是随机变量的函数
- **输出层**：预测概率是联合分布的参数化

```python
import numpy as np

# 多维随机变量示例：图像的 RGB 通道
# 假设每张 32x32 图像有 3 个通道
image = np.random.randint(0, 256, (32, 32, 3))  # RGB 图像

# 展平为随机向量
random_vector = image.flatten()
print(f"图像形状: {image.shape}")
print(f"随机向量维度: {random_vector.shape}")
print(f"前 10 个值: {random_vector[:10]}")

# 批量图像 → 随机矩阵
batch_size = 64
batch_images = np.random.randint(0, 256, (batch_size, 32, 32, 3))
batch_vectors = batch_images.reshape(batch_size, -1)
print(f"\n批量图像: {batch_images.shape}")
print(f"批量随机向量: {batch_vectors.shape}")
```

## 联合分布

### 定义

**联合分布**描述多个随机变量同时取值的概率分布。

### 离散情况

对于离散随机变量 $X$ 和 $Y$，**联合概率质量函数 (Joint PMF)**：

$$
p(x, y) = P(X = x, Y = y)
$$

**性质**：
- $p(x, y) \geq 0$
- $\displaystyle\sum_x \sum_y p(x, y) = 1$

### 连续情况

对于连续随机变量 $X$ 和 $Y$，**联合概率密度函数 (Joint PDF)**：

$$
P((X, Y) \in A) = \iint_A f(x, y) \, dx \, dy
$$

**性质**：
- $f(x, y) \geq 0$
- $\displaystyle\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(x, y) \, dx \, dy = 1$

### 二维正态分布

最重要的二维连续分布：

$$
f(x, y) = \frac{1}{2\pi\sigma_X\sigma_Y\sqrt{1-\rho^2}} \exp\left(-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_X)^2}{\sigma_X^2} - \frac{2\rho(x-\mu_X)(y-\mu_Y)}{\sigma_X\sigma_Y} + \frac{(y-\mu_Y)^2}{\sigma_Y^2}\right]\right)
$$

其中 $\rho$ 是相关系数。

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import multivariate_normal

# 二维正态分布
mu = [0, 0]
cov = [[1, 0.5], [0.5, 1]]  # 协方差矩阵

# 创建网格
x, y = np.mgrid[-3:3:.01, -3:3:.01]
pos = np.dstack((x, y))

# 计算联合 PDF
rv = multivariate_normal(mu, cov)
pdf = rv.pdf(pos)

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(12, 5))

# 等高线图
contour = axes[0].contourf(x, y, pdf, levels=20, cmap='viridis')
axes[0].set_xlabel('X')
axes[0].set_ylabel('Y')
axes[0].set_title('二维正态分布 Joint PDF (等高线)')
plt.colorbar(contour, ax=axes[0])

# 3D 曲面图
ax3d = fig.add_subplot(122, projection='3d')
ax3d.plot_surface(x, y, pdf, cmap='viridis', alpha=0.8)
ax3d.set_xlabel('X')
ax3d.set_ylabel('Y')
ax3d.set_zlabel('f(x,y)')
ax3d.set_title('二维正态分布 3D 视图')

plt.tight_layout()
plt.savefig('joint_distribution.png', dpi=100)
print("联合分布图像已保存")

# 离散联合分布示例
print("\n离散联合分布示例:")
print("="*40)

# 掷两个骰子
die1 = [1, 2, 3, 4, 5, 6]
die2 = [1, 2, 3, 4, 5, 6]

# 联合 PMF 表 (每个组合概率为 1/36)
print("P(X=x, Y=y) 表格 (部分):")
print(f"P(1,1) = 1/36 ≈ {1/36:.4f}")
print(f"P(1,2) = 1/36 ≈ {1/36:.4f}")
print(f"P(6,6) = 1/36 ≈ {1/36:.4f}")
print(f"所有概率之和: {6*6 * 1/36:.1f}")
```

## 边缘分布

### 定义

**边缘分布**是从联合分布中"提取"单个随机变量的分布，通过"边缘化"（求和或积分）其他变量得到。

### 离散情况

$$
p_X(x) = \sum_y p(x, y)
$$

$$
p_Y(y) = \sum_x p(x, y)
$$

### 连续情况

$$
f_X(x) = \int_{-\infty}^{\infty} f(x, y) \, dy
$$

$$
f_Y(y) = \int_{-\infty}^{\infty} f(x, y) \, dx
$$

### 直观理解

想象联合分布是一个表格，边缘分布就是对行/列求和得到的"边缘" totals。

```python
import numpy as np
import pandas as pd

# 离散联合分布示例
# 两个骰子的联合分布
joint_pmf = np.ones((6, 6)) / 36  # 每个格子 1/36

# 计算边缘分布
marginal_X = joint_pmf.sum(axis=1)  # 按列求和
marginal_Y = joint_pmf.sum(axis=0)  # 按行求和

print("联合分布表格 (骰子1 × 骰子2):")
df = pd.DataFrame(joint_pmf, 
                  index=[f'X={i}' for i in range(1, 7)],
                  columns=[f'Y={i}' for i in range(1, 7)])
print(df.round(4))
print(f"\nX 的边缘分布: {marginal_X}")
print(f"Y 的边缘分布: {marginal_Y}")

# 验证归一化
print(f"\nX 边缘分布之和: {marginal_X.sum():.4f}")
print(f"Y 边缘分布之和: {marginal_Y.sum():.4f}")

# 另一个示例：非独立的联合分布
print("\n" + "="*50)
print("非独立联合分布示例:")

# 假设联合分布
joint_custom = np.array([
    [0.1, 0.1, 0.05],
    [0.15, 0.2, 0.1],
    [0.1, 0.15, 0.05]
])

marginal_X_custom = joint_custom.sum(axis=1)
marginal_Y_custom = joint_custom.sum(axis=0)

print("联合 PMF:")
print(joint_custom)
print(f"\nX 的边缘分布: {marginal_X_custom}")
print(f"Y 的边缘分布: {marginal_Y_custom}")
```

## 条件分布

### 定义

**条件分布**是在已知一个变量取值的情况下，另一个变量的概率分布。

### 条件 PMF（离散）

$$
p_{Y|X}(y|x) = \frac{p(x, y)}{p_X(x)}, \quad p_X(x) > 0
$$

### 条件 PDF（连续）

$$
f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}, \quad f_X(x) > 0
$$

### 贝叶斯公式的分布形式

$$
f_{X|Y}(x|y) = \frac{f_{Y|X}(y|x) f_X(x)}{f_Y(y)}
$$

### 条件期望

$$
\mathbb{E}[Y|X = x] = \sum_y y \cdot p_{Y|X}(y|x) \quad \text{（离散）}
$$

$$
\mathbb{E}[Y|X = x] = \int_{-\infty}^{\infty} y \cdot f_{Y|X}(y|x) \, dy \quad \text{（连续）}
$$

```python
import numpy as np

# 条件分布示例
# 联合分布
joint = np.array([
    [0.1, 0.1, 0.05],   # X=0
    [0.15, 0.2, 0.1],   # X=1
    [0.1, 0.15, 0.05]   # X=2
])

# 边缘分布
marginal_X = joint.sum(axis=1)
marginal_Y = joint.sum(axis=0)

print("条件分布计算:")
print("="*40)

# 条件分布 P(Y|X=1)
x_val = 1
conditional_Y_given_X1 = joint[x_val, :] / marginal_X[x_val]
print(f"边缘分布 P(X={x_val}) = {marginal_X[x_val]:.2f}")
print(f"条件分布 P(Y|X={x_val}): {conditional_Y_given_X1}")
print(f"条件分布之和: {conditional_Y_given_X1.sum():.4f}")

# 计算条件期望 E[Y|X=1]
y_values = np.array([0, 1, 2])
conditional_expectation = np.sum(y_values * conditional_Y_given_X1)
print(f"条件期望 E[Y|X={x_val}] = {conditional_expectation:.4f}")

# 所有条件期望
print("\n所有条件期望:")
for x in range(3):
    cond_dist = joint[x, :] / marginal_X[x]
    cond_exp = np.sum(y_values * cond_dist)
    print(f"E[Y|X={x}] = {cond_exp:.4f}")
```

## 独立性

### 定义

随机变量 $X$ 和 $Y$ **独立**当且仅当联合分布等于边缘分布的乘积：

$$
f(x, y) = f_X(x) \cdot f_Y(y)
$$

或等价地：

$$
P(X \in A, Y \in B) = P(X \in A) \cdot P(Y \in B)
$$

### 独立的等价条件

以下条件等价：
1. $f(x, y) = f_X(x) f_Y(y)$
2. $F(x, y) = F_X(x) F_Y(y)$（联合 CDF = 边缘 CDF 之积）
3. $f_{Y|X}(y|x) = f_Y(y)$（条件分布 = 边缘分布）
4. $\mathbb{E}[g(X)h(Y)] = \mathbb{E}[g(X)] \mathbb{E}[h(Y)]$（对所有函数 g, h）

### 独立 vs 不相关

| 概念 | 定义 | 强弱 |
|------|------|------|
| 独立 | $f(x,y) = f_X(x)f_Y(y)$ | 强 |
| 不相关 | $\text{Cov}(X,Y) = 0$ | 弱 |

**重要结论**：
- 独立 ⇒ 不相关
- 不相关 ⇏ 独立

```python
import numpy as np

# 独立性检验示例
print("独立性检验:")
print("="*50)

# 示例1：独立的联合分布
joint_independent = np.outer([0.3, 0.5, 0.2], [0.4, 0.6])
marginal_X = joint_independent.sum(axis=1)
marginal_Y = joint_independent.sum(axis=0)

product = np.outer(marginal_X, marginal_Y)

print("示例1: 检验独立性")
print(f"联合分布:\n{joint_independent}")
print(f"\n边缘分布之积:\n{product}")
print(f"差异: {np.abs(joint_independent - product).sum():.10f}")
print(f"独立? {np.allclose(joint_independent, product)}")

# 示例2：不独立的联合分布
joint_dependent = np.array([
    [0.1, 0.1, 0.05],
    [0.15, 0.2, 0.1],
    [0.1, 0.15, 0.05]
])
marginal_X2 = joint_dependent.sum(axis=1)
marginal_Y2 = joint_dependent.sum(axis=0)
product2 = np.outer(marginal_X2, marginal_Y2)

print("\n示例2: 检验独立性")
print(f"联合分布:\n{joint_dependent}")
print(f"\n边缘分布之积:\n{product2}")
print(f"差异: {np.abs(joint_dependent - product2).sum():.4f}")
print(f"独立? {np.allclose(joint_dependent, product2)}")

# 不相关但不独立的经典例子
print("\n" + "="*50)
print("不相关但不独立的例子: X ~ Uniform(-1,1), Y = X²")

n = 100000
X = np.random.uniform(-1, 1, n)
Y = X**2

# 计算协方差
cov_XY = np.cov(X, Y)[0, 1]
corr_XY = np.corrcoef(X, Y)[0, 1]

print(f"Cov(X, Y) = {cov_XY:.6f}")
print(f"Corr(X, Y) = {corr_XY:.6f}")
print(f"不相关? {abs(corr_XY) < 0.01}")
print(f"独立? False (Y 完全由 X 决定: Y = X²)")
```

## 期望的性质

### 基本性质回顾

**期望的线性**（最重要的性质）：

$$
\mathbb{E}[aX + bY] = a\mathbb{E}[X] + b\mathbb{E}[Y]
$$

**常数期望**：

$$
\mathbb{E}[c] = c
$$

### 函数的期望

$$
\mathbb{E}[g(X)] = \sum_x g(x) \cdot p(x) \quad \text{（离散）}
$$

$$
\mathbb{E}[g(X)] = \int_{-\infty}^{\infty} g(x) \cdot f(x) \, dx \quad \text{（连续）}
$$

### 矩

**k 阶原点矩**：

$$
\mathbb{E}[X^k]
$$

**k 阶中心矩**：

$$
\mathbb{E}[(X - \mathbb{E}[X])^k]
$$

| 矩 | 名称 | 含义 |
|----|------|------|
| $\mathbb{E}[X]$ | 一阶原点矩 | 期望（位置） |
| $\mathbb{E}[X^2]$ | 二阶原点矩 | 能量 |
| $\mathbb{E}[(X-\mu)^2]$ | 二阶中心矩 | 方差（离散程度） |
| $\mathbb{E}[(X-\mu)^3]$ | 三阶中心矩 | 偏度（对称性） |
| $\mathbb{E}[(X-\mu)^4]$ | 四阶中心矩 | 峰度（尾部厚度） |

### 乘积期望

一般情况下：

$$
\mathbb{E}[XY] \neq \mathbb{E}[X]\mathbb{E}[Y]
$$

当 $X$ 和 $Y$ **独立**时：

$$
\mathbb{E}[XY] = \mathbb{E}[X]\mathbb{E}[Y]
$$

```python
import numpy as np

# 期望性质演示
print("期望的性质:")
print("="*50)

# 创建随机变量
np.random.seed(42)
X = np.random.normal(5, 2, 100000)
Y = np.random.normal(3, 1, 100000)

# 线性性质
a, b = 2, 3
lhs = (a * X + b * Y).mean()
rhs = a * X.mean() + b * Y.mean()
print(f"线性性质: E[{a}X + {b}Y]")
print(f"  直接计算: {lhs:.4f}")
print(f"  分解计算: {rhs:.4f}")
print(f"  相等? {np.isclose(lhs, rhs)}")

# 矩的计算
print(f"\n矩的计算:")
print(f"  E[X] = {X.mean():.4f}")
print(f"  E[X²] = {(X**2).mean():.4f}")
print(f"  E[(X-μ)²] = {((X - X.mean())**2).mean():.4f}")
print(f"  Var(X) = {X.var():.4f}")

# 验证 Var(X) = E[X²] - E[X]²
calculated_var = (X**2).mean() - X.mean()**2
print(f"  验证: E[X²] - E[X]² = {calculated_var:.4f}")

# 独立变量的乘积期望
Z = np.random.normal(0, 1, 100000)  # 与 X 独立
print(f"\n独立变量乘积期望:")
print(f"  E[X] = {X.mean():.4f}")
print(f"  E[Z] = {Z.mean():.4f}")
print(f"  E[X·Z] = {(X * Z).mean():.4f}")
print(f"  E[X]·E[Z] = {X.mean() * Z.mean():.4f}")
```

## 协方差

### 定义

**协方差**衡量两个随机变量的**线性关系**：

$$
\text{Cov}(X, Y) = \mathbb{E}[(X - \mathbb{E}[X])(Y - \mathbb{E}[Y])]
$$

### 计算公式

$$
\text{Cov}(X, Y) = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y]
$$

### 协方差的性质

1. **对称性**：$\text{Cov}(X, Y) = \text{Cov}(Y, X)$
2. **自身协方差**：$\text{Cov}(X, X) = \text{Var}(X)$
3. **常数**：$\text{Cov}(X, c) = 0$
4. **线性性**：$\text{Cov}(aX, bY) = ab \cdot \text{Cov}(X, Y)$
5. **可加性**：$\text{Cov}(X + Y, Z) = \text{Cov}(X, Z) + \text{Cov}(Y, Z)$
6. **独立**：若 $X, Y$ 独立，则 $\text{Cov}(X, Y) = 0$

### 和的方差

$$
\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)
$$

**和的方差公式推导**：

**第一步**：利用方差的定义。

$$\text{Var}(X + Y) = \mathbb{E}[(X + Y - \mathbb{E}[X + Y])^2]$$

**第二步**：利用期望的线性性质展开。

$$\mathbb{E}[X + Y] = \mathbb{E}[X] + \mathbb{E}[Y] = \mu_X + \mu_Y$$

因此：

$$\text{Var}(X + Y) = \mathbb{E}[(X - \mu_X + Y - \mu_Y)^2]$$

**第三步**：展开平方项。

$$= \mathbb{E}[(X - \mu_X)^2 + (Y - \mu_Y)^2 + 2(X - \mu_X)(Y - \mu_Y)]$$

**第四步**：利用期望的线性性质分离各项。

$$= \mathbb{E}[(X - \mu_X)^2] + \mathbb{E}[(Y - \mu_Y)^2] + 2\mathbb{E}[(X - \mu_X)(Y - \mu_Y)]$$

**第五步**：识别各项。

- $\mathbb{E}[(X - \mu_X)^2] = \text{Var}(X)$
- $\mathbb{E}[(Y - \mu_Y)^2] = \text{Var}(Y)$
- $\mathbb{E}[(X - \mu_X)(Y - \mu_Y)] = \text{Cov}(X, Y)$

**第六步**：综合结果。

$$\boxed{\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)}$$

**推论**：当 $X, Y$ 独立时，$\text{Cov}(X, Y) = 0$，因此：

$$\boxed{\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y)}$$

**一般形式**：对于 $n$ 个随机变量：

$$\text{Var}\left(\sum_{i=1}^n X_i\right) = \sum_{i=1}^n \text{Var}(X_i) + 2\sum_{i < j} \text{Cov}(X_i, X_j)$$

```python
import numpy as np

# 协方差计算示例
np.random.seed(42)
n = 100000

# 正相关
X1 = np.random.normal(0, 1, n)
Y1 = X1 + np.random.normal(0, 0.5, n)  # Y 与 X 正相关

# 负相关
X2 = np.random.normal(0, 1, n)
Y2 = -X2 + np.random.normal(0, 0.5, n)  # Y 与 X 负相关

# 不相关
X3 = np.random.normal(0, 1, n)
Y3 = np.random.normal(0, 1, n)  # 独立

# 计算协方差
cov1 = np.cov(X1, Y1)[0, 1]
cov2 = np.cov(X2, Y2)[0, 1]
cov3 = np.cov(X3, Y3)[0, 1]

print("协方差示例:")
print("="*50)
print(f"正相关: Cov(X1, Y1) = {cov1:.4f}")
print(f"负相关: Cov(X2, Y2) = {cov2:.4f}")
print(f"不相关: Cov(X3, Y3) = {cov3:.4f}")

# 手动计算验证
def manual_cov(X, Y):
    return ((X - X.mean()) * (Y - Y.mean())).mean()

print(f"\n手动计算验证:")
print(f"正相关: {manual_cov(X1, Y1):.4f}")
print(f"负相关: {manual_cov(X2, Y2):.4f}")
print(f"不相关: {manual_cov(X3, Y3):.4f}")

# 和的方差公式验证
print(f"\n和的方差公式验证:")
print(f"Var(X1) = {X1.var():.4f}")
print(f"Var(Y1) = {Y1.var():.4f}")
print(f"Var(X1+Y1) = {(X1+Y1).var():.4f}")
print(f"Var(X1) + Var(Y1) + 2*Cov(X1,Y1) = {X1.var() + Y1.var() + 2*cov1:.4f}")
```

## 相关系数

### 定义

**相关系数**是标准化的协方差，范围在 $[-1, 1]$ 之间：

$$
\rho_{XY} = \frac{\text{Cov}(X, Y)}{\sqrt{\text{Var}(X) \cdot \text{Var}(Y)}} = \frac{\text{Cov}(X, Y)}{\sigma_X \sigma_Y}
$$

### 相关系数的性质

1. **范围**：$-1 \leq \rho \leq 1$
2. **无量纲**：消除量纲影响
3. **对称性**：$\rho_{XY} = \rho_{YX}$
4. **线性变换**：$\rho_{aX+b, cY+d} = \text{sign}(ac) \cdot \rho_{XY}$

### 相关系数的解释

| $\rho$ 值 | 关系 |
|-----------|------|
| $\rho = 1$ | 完全正线性相关 |
| $0 < \rho < 1$ | 正相关 |
| $\rho = 0$ | 不相关（无线性关系） |
| $-1 < \rho < 0$ | 负相关 |
| $\rho = -1$ | 完全负线性相关 |

### ⚠️ 重要注意

$\rho = 0$ 只意味着**没有线性关系**，不代表没有非线性关系！

```python
import numpy as np
```

## 协方差矩阵

### 定义

对于随机向量 $\mathbf{X} = (X_1, X_2, \ldots, X_n)^\top$，**协方差矩阵**定义为：

$$
\mathbf{\Sigma} = \text{Cov}(\mathbf{X}) = \mathbb{E}[(\mathbf{X} - \boldsymbol{\mu})(\mathbf{X} - \boldsymbol{\mu})^\top]
$$

其中 $\boldsymbol{\mu} = \mathbb{E}[\mathbf{X}]$。

### 矩阵形式

$$
\mathbf{\Sigma} = \begin{bmatrix}
\text{Var}(X_1) & \text{Cov}(X_1, X_2) & \cdots & \text{Cov}(X_1, X_n) \\
\text{Cov}(X_2, X_1) & \text{Var}(X_2) & \cdots & \text{Cov}(X_2, X_n) \\
\vdots & \vdots & \ddots & \vdots \\
\text{Cov}(X_n, X_1) & \text{Cov}(X_n, X_2) & \cdots & \text{Var}(X_n)
\end{bmatrix}
$$

### 协方差矩阵的性质

1. **对称**：$\mathbf{\Sigma} = \mathbf{\Sigma}^\top$
2. **半正定**：$\mathbf{a}^\top \mathbf{\Sigma} \mathbf{a} \geq 0$ 对任意 $\mathbf{a}$
3. **对角元非负**：$\Sigma_{ii} = \text{Var}(X_i) \geq 0$
4. **特征值非负**

### 样本协方差矩阵

给定 $m$ 个样本 $\mathbf{x}_1, \ldots, \mathbf{x}_m$，样本协方差矩阵：

$$
\hat{\mathbf{\Sigma}} = \frac{1}{m-1} \sum_{i=1}^m (\mathbf{x}_i - \bar{\mathbf{x}})(\mathbf{x}_i - \bar{\mathbf{x}})^\top
$$

```python
import numpy as np

# 协方差矩阵计算
np.random.seed(42)

# 生成相关数据
n_samples = 1000
n_features = 3

# 真实协方差矩阵
true_cov = np.array([
    [1.0, 0.8, 0.3],
    [0.8, 2.0, 0.5],
    [0.3, 0.5, 1.5]
])

print("真实协方差矩阵:")
print(true_cov)

# 从多元正态分布采样
mean = np.zeros(n_features)
samples = np.random.multivariate_normal(mean, true_cov, n_samples)

print(f"\n样本形状: {samples.shape}")

# 计算样本协方差矩阵
sample_cov = np.cov(samples.T)
print("\n样本协方差矩阵:")
print(sample_cov)

# 协方差矩阵的性质验证
print("\n协方差矩阵性质验证:")
print(f"对称? {np.allclose(sample_cov, sample_cov.T)}")
print(f"特征值: {np.linalg.eigvalsh(sample_cov)}")
print(f"半正定? {np.all(np.linalg.eigvalsh(sample_cov) >= 0)}")

# 手动计算协方差矩阵
def manual_cov_matrix(X):
    """手动计算协方差矩阵"""
    X_centered = X - X.mean(axis=0)
    return X_centered.T @ X_centered / (X.shape[0] - 1)

manual_cov = manual_cov_matrix(samples)
print(f"\n手动计算结果与 np.cov 一致? {np.allclose(sample_cov, manual_cov)}")
```

## 高阶矩：偏度与峰度

### 偏度 (Skewness)

**偏度**衡量分布的**不对称性**：

$$
\gamma_1 = \mathbb{E}\left[\left(\frac{X - \mu}{\sigma}\right)^3\right] = \frac{\mathbb{E}[(X - \mu)^3]}{\sigma^3}
$$

| 偏度 | 分布形状 |
|------|----------|
| $\gamma_1 = 0$ | 对称 |
| $\gamma_1 > 0$ | 右偏（正偏），长尾在右 |
| $\gamma_1 < 0$ | 左偏（负偏），长尾在左 |

### 峰度 (Kurtosis)

**峰度**衡量分布的**尾部厚度**（相对于正态分布）：

$$
\gamma_2 = \mathbb{E}\left[\left(\frac{X - \mu}{\sigma}\right)^4\right] - 3
$$

**超额峰度**（减去3使正态分布为0）：

| 峰度 | 分布特征 |
|------|----------|
| $\gamma_2 = 0$ | 正态分布 |
| $\gamma_2 > 0$ | 尖峰（厚尾），重尾分布 |
| $\gamma_2 < 0$ | 平顶（薄尾），轻尾分布 |

```python
import numpy as np
```

## 在深度学习中的应用

### 1. Batch Normalization

Batch Normalization 使用批统计量估计总体统计量：

$$
\hat{x}_i = \frac{x_i - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}
$$

其中 $\mu_B$ 和 $\sigma_B^2$ 是小批量的均值和方差。

```python
import numpy as np

class BatchNorm:
    """简化的 Batch Normalization 实现"""
    
    def __init__(self, num_features, eps=1e-5, momentum=0.1):
        self.eps = eps
        self.momentum = momentum
        
        # 可学习参数
        self.gamma = np.ones(num_features)
        self.beta = np.zeros(num_features)
        
        # 运行时统计量
        self.running_mean = np.zeros(num_features)
        self.running_var = np.ones(num_features)
        
        self.training = True
    
    def forward(self, x):
        """
        Parameters:
        -----------
        x : array, shape (batch_size, num_features)
        """
        if self.training:
            # 计算批统计量
            batch_mean = x.mean(axis=0)
            batch_var = x.var(axis=0)
            
            # 更新运行时统计量（移动平均）
            self.running_mean = (1 - self.momentum) * self.running_mean + self.momentum * batch_mean
            self.running_var = (1 - self.momentum) * self.running_var + self.momentum * batch_var
            
            # 标准化
            x_norm = (x - batch_mean) / np.sqrt(batch_var + self.eps)
        else:
            # 使用运行时统计量
            x_norm = (x - self.running_mean) / np.sqrt(self.running_var + self.eps)
        
        # 缩放和平移
        return self.gamma * x_norm + self.beta

# 示例
bn = BatchNorm(num_features=64)
batch = np.random.randn(32, 64) * 2 + 1  # 非标准化的输入

output = bn.forward(batch)
print("Batch Normalization 示例:")
print(f"输入: 均值={batch.mean():.4f}, 方差={batch.var():.4f}")
print(f"输出: 均值={output.mean():.4f}, 方差={output.var():.4f}")
```

### 2. 协方差矩阵在 PCA 中的应用

```python
import numpy as np

def pca(X, n_components):
    """
    PCA 降维
    
    Parameters:
    -----------
    X : array, shape (n_samples, n_features)
    n_components : int
        保留的主成分数量
    """
    # 中心化
    X_centered = X - X.mean(axis=0)
    
    # 计算协方差矩阵
    cov_matrix = np.cov(X_centered.T)
    
    # 特征值分解
    eigenvalues, eigenvectors = np.linalg.eigh(cov_matrix)
    
    # 按特征值降序排列
    idx = np.argsort(eigenvalues)[::-1]
    eigenvalues = eigenvalues[idx]
    eigenvectors = eigenvectors[:, idx]
    
    # 选择前 n_components 个主成分
    principal_components = eigenvectors[:, :n_components]
    
    # 投影
    X_pca = X_centered @ principal_components
    
    return X_pca, eigenvalues[:n_components]

# 示例
np.random.seed(42)
X = np.random.randn(1000, 10)

# 添加一些相关性
X[:, 1] = X[:, 0] + np.random.randn(1000) * 0.1
X[:, 2] = X[:, 0] + X[:, 1] + np.random.randn(1000) * 0.1

X_pca, explained_var = pca(X, n_components=3)

print("PCA 示例:")
print(f"原始数据形状: {X.shape}")
print(f"降维后形状: {X_pca.shape}")
print(f"前3个主成分解释的方差: {explained_var}")
print(f"方差解释比例: {explained_var / explained_var.sum()}")
```

### 3. Layer Normalization

```python
import numpy as np

class LayerNorm:
    """Layer Normalization 实现"""
    
    def __init__(self, normalized_shape, eps=1e-5):
        self.eps = eps
        self.gamma = np.ones(normalized_shape)
        self.beta = np.zeros(normalized_shape)
    
    def forward(self, x):
        """
        Parameters:
        -----------
        x : array, shape (batch_size, seq_len, hidden_size) 或 (batch_size, hidden_size)
        """
        # 在最后一个维度上计算均值和方差
        mean = x.mean(axis=-1, keepdims=True)
        var = x.var(axis=-1, keepdims=True)
        
        # 标准化
        x_norm = (x - mean) / np.sqrt(var + self.eps)
        
        # 缩放和平移
        return self.gamma * x_norm + self.beta

# 示例
ln = LayerNorm(normalized_shape=512)
x = np.random.randn(32, 10, 512)  # (batch, seq_len, hidden)

output = ln.forward(x)
print("Layer Normalization 示例:")
print(f"输入形状: {x.shape}")
print(f"输出形状: {output.shape}")
print(f"每个样本的均值: {output[0, 0, :].mean():.6f}")  # 接近0
print(f"每个样本的方差: {output[0, 0, :].var():.6f}")  # 接近1
```

## 小结

本章介绍了多维随机变量的核心概念，这些是理解深度学习中数据表示和特征工程的基础。

### 核心概念对照表

| 概念 | 公式 | 深度学习应用 |
|------|------|--------------|
| 联合分布 | $p(x,y) = P(X=x, Y=y)$ | 多特征建模 |
| 边缘分布 | $p_X(x) = \sum_y p(x,y)$ | 特征提取 |
| 条件分布 | $p(y|x) = p(x,y)/p_X(x)$ | 生成模型 |
| 协方差 | $\text{Cov}(X,Y) = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y]$ | 特征相关性 |
| 相关系数 | $\rho = \text{Cov}(X,Y)/(\sigma_X \sigma_Y)$ | 特征选择 |
| 协方差矩阵 | $\mathbf{\Sigma} = \mathbb{E}[(\mathbf{X}-\boldsymbol{\mu})(\mathbf{X}-\boldsymbol{\mu})^\top]$ | PCA, BatchNorm |

### 关键要点

1. **联合分布**：描述多个变量的整体分布
2. **边缘分布**：从联合分布提取单个变量分布
3. **条件分布**：给定一个变量，另一个变量的分布
4. **协方差**：衡量线性相关性，单位敏感
5. **相关系数**：标准化的协方差，$[-1, 1]$ 范围
6. **独立 vs 不相关**：独立 ⇒ 不相关，但反之不成立
7. **协方差矩阵**：高维数据的核心工具，对称半正定

### 深度学习中的核心应用

| 技术 | 使用的概率概念 |
|------|---------------|
| Batch Normalization | 均值、方差估计 |
| Layer Normalization | 层级统计量 |
| PCA | 协方差矩阵特征分解 |
| Dropout | 伯努利采样独立性 |
| Attention | 条件概率分布 |
