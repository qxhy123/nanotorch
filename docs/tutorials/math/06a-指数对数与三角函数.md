## 指数函数

### 定义

**指数函数**的泰勒级数展开：

$$
e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \cdots
$$

### 核心性质

1. **基本恒等式**：

$$
e^{x+y} = e^x \cdot e^y
$$

**证明**：

**第一步**：考虑 $f(x) = e^{x+y}$（固定 $y$）。

$$\frac{d}{dx} e^{x+y} = e^{x+y}$$

**第二步**：考虑 $g(x) = e^x \cdot e^y$。

$$\frac{d}{dx}(e^x \cdot e^y) = e^x \cdot e^y$$

**第三步**：两个函数有相同的导数，因此只差一个常数。

令 $x = 0$：

$$f(0) = e^y = g(0) = e^0 \cdot e^y = e^y$$

所以 $f(x) = g(x)$ 对所有 $x$ 成立。

$$\boxed{e^{x+y} = e^x \cdot e^y}$$

2. **导数**：

$$
\frac{d}{dx} e^x = e^x
$$

这是唯一一个导数等于自身的非零函数！

**指数函数导数的推导**：

**方法一：从定义出发**

$$\frac{d}{dx} e^x = \lim_{h \to 0} \frac{e^{x+h} - e^x}{h} = \lim_{h \to 0} \frac{e^x \cdot e^h - e^x}{h} = e^x \lim_{h \to 0} \frac{e^h - 1}{h}$$

关键是证明 $\lim_{h \to 0} \frac{e^h - 1}{h} = 1$。

**方法二：使用反函数求导**

设 $y = e^x$，则 $x = \ln y$。

$$\frac{dy}{dx} = \frac{1}{\frac{dx}{dy}} = \frac{1}{\frac{1}{y}} = y = e^x$$

$$\boxed{\frac{d}{dx} e^x = e^x}$$

3. **恒正性**：$e^x > 0, \forall x \in \mathbb{R}$

4. **单调性**：$e^x$ 严格单调递增

5. **极限行为**：
   - $\lim_{x \to -\infty} e^x = 0$
   - $\lim_{x \to +\infty} e^x = +\infty$

6. **对称性**：

$$
e^{-x} = \frac{1}{e^x}
$$

### 广义指数函数

对于任意底数 $a > 0, a \neq 1$：

$$
a^x = e^{x \ln a}
$$

**性质**：
- $\frac{d}{dx} a^x = a^x \ln a$
- $a^{x+y} = a^x \cdot a^y$

## 对数函数

### 定义

**自然对数**是指数函数的反函数：

$$
y = \ln x \iff x = e^y, \quad x > 0
$$

### 核心性质

1. **基本恒等式**：
   - $\ln(xy) = \ln x + \ln y$
   - $\ln(x/y) = \ln x - \ln y$
   - $\ln(x^n) = n \ln x$

2. **导数**：

$$
\frac{d}{dx} \ln x = \frac{1}{x}
$$

3. **复合函数导数**：

$$
\frac{d}{dx} \ln f(x) = \frac{f'(x)}{f(x)}
$$

4. **极限行为**：
   - $\lim_{x \to 0^+} \ln x = -\infty$
   - $\lim_{x \to +\infty} \ln x = +\infty$

5. **换底公式**：

$$
\log_a x = \frac{\ln x}{\ln a}
$$

### 泰勒级数展开

对于 $|x| < 1$：

$$
\ln(1 + x) = \sum_{n=1}^{\infty} \frac{(-1)^{n+1} x^n}{n} = x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots
$$

### 指数与对数在深度学习中的角色

**1. 概率表示（Softmax）**

$$
P(y=i) = \frac{e^{z_i}}{\sum_{j} e^{z_j}}
$$

**2. 对数似然**

$$
\mathcal{L}(\theta) = \sum_{i=1}^N \ln P(y_i | x_i; \theta)
$$

**3. 指数移动平均**

$$
\mathbf{v}_t = \beta \mathbf{v}_{t-1} + (1-\beta) \mathbf{g}_t
$$

### 数值稳定的实现

```python
import numpy as np

def log_sum_exp(x, axis=None):
    """
    数值稳定的 log-sum-exp
    
    log(sum(exp(x))) = max(x) + log(sum(exp(x - max(x))))
    """
    x_max = np.max(x, axis=axis, keepdims=True)
    result = x_max + np.log(np.sum(np.exp(x - x_max), axis=axis, keepdims=True))
    return np.squeeze(result, axis=axis) if axis is not None else result

def softmax(x, axis=-1):
    """数值稳定的 softmax"""
    x_max = np.max(x, axis=axis, keepdims=True)
    exp_x = np.exp(x - x_max)
    return exp_x / np.sum(exp_x, axis=axis, keepdims=True)

def log_softmax(x, axis=-1):
    """数值稳定的 log-softmax"""
    return x - log_sum_exp(x, axis=axis, keepdims=True)

# 示例
x = np.array([1.0, 2.0, 3.0])
print(f"Softmax: {softmax(x)}")
print(f"Log-softmax: {log_softmax(x)}")
print(f"Log-sum-exp: {log_sum_exp(x)}")
```

## 三角函数

### 基本定义

在单位圆上，角度 $\theta$ 对应：

$$
\begin{align}
\sin \theta &= \frac{\text{对边}}{\text{斜边}} = \frac{y}{r} \\[6pt]
\cos \theta &= \frac{\text{邻边}}{\text{斜边}} = \frac{x}{r} \\[6pt]
\tan \theta &= \frac{\sin \theta}{\cos \theta} = \frac{y}{x}
\end{align}
$$

### 核心恒等式

**勾股定理**：

$$
\sin^2 \theta + \cos^2 \theta = 1
$$

**和差公式**：

$$
\begin{align}
\sin(a \pm b) &= \sin a \cos b \pm \cos a \sin b \\[6pt]
\cos(a \pm b) &= \cos a \cos b \mp \sin a \sin b
\end{align}
$$

**倍角公式**：

$$
\begin{align}
\sin 2\theta &= 2 \sin \theta \cos \theta \\[6pt]
\cos 2\theta &= \cos^2 \theta - \sin^2 \theta = 2\cos^2 \theta - 1 = 1 - 2\sin^2 \theta
\end{align}
$$

### 导数

$$
\frac{d}{dx} \sin x = \cos x
$$

$$
\frac{d}{dx} \cos x = -\sin x
$$

$$
\frac{d}{dx} \tan x = \sec^2 x = \frac{1}{\cos^2 x}
$$

### 周期性

- $\sin(x + 2\pi) = \sin x$
- $\cos(x + 2\pi) = \cos x$
- $\tan(x + \pi) = \tan x$

## 双曲函数

### 定义

**双曲正弦**：

$$
\sinh x = \frac{e^x - e^{-x}}{2}
$$

**双曲余弦**：

$$
\cosh x = \frac{e^x + e^{-x}}{2}
$$

**双曲正切**：

$$
\tanh x = \frac{\sinh x}{\cosh x} = \frac{e^x - e^{-x}}{e^x + e^{-x}}
$$

### 核心恒等式

**双曲勾股定理**：

$$
\cosh^2 x - \sinh^2 x = 1
$$

**和差公式**：

$$
\begin{align}
\sinh(a \pm b) &= \sinh a \cosh b \pm \cosh a \sinh b \\[6pt]
\cosh(a \pm b) &= \cosh a \cosh b \pm \sinh a \sinh b
\end{align}
$$

### 导数

$$
\frac{d}{dx} \sinh x = \cosh x
$$

$$
\frac{d}{dx} \cosh x = \sinh x
$$

$$
\frac{d}{dx} \tanh x = 1 - \tanh^2 x = \text{sech}^2 x
$$

### Tanh 激活函数

**与 Sigmoid 的关系**：

$$
\tanh(x) = 2\sigma(2x) - 1
$$

其中 $\sigma$ 是 Sigmoid 函数。

**性质**：
- 输出范围：$(-1, 1)$
- 零中心（比 Sigmoid 更好）
- 梯度：$\frac{d}{dx} \tanh(x) = 1 - \tanh^2(x)$
- 梯度消失问题：当 $|x|$ 很大时，梯度趋近于 0

### Tanh 与 Sigmoid 的比较

| 特性 | Sigmoid | Tanh |
|------|---------|------|
| 输出范围 | $(0, 1)$ | $(-1, 1)$ |
| 零中心 | 否 | 是 |
| 梯度范围 | $(0, 0.25]$ | $(0, 1]$ |
| 用途 | 二分类输出 | RNN 隐藏层 |

```python
import numpy as np
import matplotlib.pyplot as plt

# 可视化
x = np.linspace(-5, 5, 1000)

fig, axes = plt.subplots(2, 2, figsize=(12, 8))
```

## 在深度学习中的应用

### Softmax 的数值稳定性

在实现 Softmax 时，直接计算 $e^{z_i}$ 可能导致数值溢出：

```python
# 危险的实现
def softmax_unstable(x):
    exp_x = np.exp(x)  # 可能溢出
    return exp_x / np.sum(exp_x)

# 数值稳定的实现
def softmax_stable(x):
    x_max = np.max(x)
    exp_x = np.exp(x - x_max)  # 减去最大值防止溢出
    return exp_x / np.sum(exp_x)
```

### 交叉熵损失

使用对数可以避免数值下溢：

```python
def cross_entropy_loss(logits, labels, eps=1e-15):
    """
    数值稳定的交叉熵损失
    
    Args:
        logits: 模型输出（未归一化）
        labels: 真实标签
    """
    # 使用 log-softmax 避免数值问题
    log_probs = log_softmax(logits, axis=-1)
    
    if labels.ndim == logits.ndim:
        # One-hot labels
        return -np.sum(labels * log_probs, axis=-1)
    else:
        # Integer labels
        return -log_probs[np.arange(len(labels)), labels]
```

### 学习率调度中的余弦函数

$$
\eta_t = \eta_{\min} + \frac{1}{2}(\eta_0 - \eta_{\min})(1 + \cos(\frac{t}{T}\pi))
$$

```python
def cosine_annealing(epoch, max_epochs, eta_max, eta_min=1e-6):
    """余弦退火学习率"""
    return eta_min + 0.5 * (eta_max - eta_min) * (1 + np.cos(np.pi * epoch / max_epochs))
```

### GELU 激活函数

使用双曲正切近似：

$$
\text{GELU}(x) \approx 0.5 x \left(1 + \tanh\left[\sqrt{\frac{2}{\pi}}\left(x + 0.044715 x^3\right)\right]\right)
$$

```python
def gelu(x):
    """GELU 激活函数（tanh 近似）"""
    return 0.5 * x * (1 + np.tanh(np.sqrt(2 / np.pi) * (x + 0.044715 * x**3)))
```

## 小结

本节介绍了深度学习中的基础数学函数：

| 函数 | 定义 | 关键性质 | 深度学习应用 |
|------|------|----------|--------------|
| $e^x$ | $\sum_{n=0}^{\infty} \frac{x^n}{n!}$ | $(e^x)' = e^x$ | Softmax、概率 |
| $\ln x$ | $e^y = x$ 的反函数 | $(\ln x)' = 1/x$ | 交叉熵、MLE |
| $\sin, \cos$ | 三角比 | 周期性 | 位置编码、余弦调度 |
| $\tanh$ | $\frac{e^x-e^{-x}}{e^x+e^{-x}}$ | 零中心、$(-1,1)$ | RNN、GELU |

**关键要点**：
- 指数和对数是概率建模的基础
- 数值稳定性是实现的痛点
- Tanh 是 RNN 的常用激活函数
- 理解这些函数有助于理解更复杂的激活函数
