## 总体与样本

### 基本概念

**总体 (Population)**：研究对象的全体，通常用随机变量 $X$ 或其分布函数 $F(x)$ 来描述。

**个体**：总体中的每个基本单位。

**样本 (Sample)**：从总体中随机抽取的 $n$ 个个体的集合，记为 $X_1, X_2, \ldots, X_n$。

**样本容量**：样本中个体的数目 $n$。

### 简单随机样本

**定义**：若样本 $X_1, X_2, \ldots, X_n$ 满足以下两个条件，则称为**简单随机样本 (SRS)**：

1. **独立性**：$X_1, X_2, \ldots, X_n$ 相互独立
2. **同分布性**：每个 $X_i$ 与总体 $X$ 具有相同的分布

即 $X_1, X_2, \ldots, X_n \stackrel{i.i.d.}{\sim} F(x)$

### 样本的双重性

样本具有**双重性**：

| 角度 | 含义 | 表示 |
|------|------|------|
| 随机变量 | 抽取前，结果是随机的 | $X_1, X_2, \ldots, X_n$ |
| 具体数值 | 抽取后，结果是确定的 | $x_1, x_2, \ldots, x_n$ |

### 样本空间

**样本空间**：所有可能的样本取值的集合，记为 $\mathcal{X}^n$。

对于连续总体，样本空间通常是 $\mathbb{R}^n$ 的子集。

```python
import numpy as np
import matplotlib.pyplot as plt

# 示例：从正态总体中抽取样本
np.random.seed(42)

# 总体参数
true_mu = 100
true_sigma = 15

# 抽取样本
n = 50  # 样本容量
sample = np.random.normal(true_mu, true_sigma, n)

print(f"总体参数: μ = {true_mu}, σ = {true_sigma}")
print(f"样本容量: n = {n}")
print(f"样本前5个值: {sample[:5]}")
print(f"样本均值: {sample.mean():.2f}")
print(f"样本标准差: {sample.std():.2f}")

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(12, 4))

# 样本直方图
axes[0].hist(sample, bins=15, density=True, alpha=0.7, edgecolor='black')
x = np.linspace(true_mu - 4*true_sigma, true_mu + 4*true_sigma, 100)
axes[0].plot(x, 1/(true_sigma*np.sqrt(2*np.pi)) * np.exp(-(x-true_mu)**2/(2*true_sigma**2)), 
             'r-', linewidth=2, label='总体分布')
axes[0].axvline(sample.mean(), color='green', linestyle='--', label=f'样本均值={sample.mean():.1f}')
axes[0].set_xlabel('值')
axes[0].set_ylabel('密度')
axes[0].set_title('样本直方图与总体分布')
axes[0].legend()
axes[0].grid(True, alpha=0.3)

# 多次抽样的均值分布
n_samples = 1000
sample_means = [np.random.normal(true_mu, true_sigma, n).mean() for _ in range(n_samples)]
axes[1].hist(sample_means, bins=30, density=True, alpha=0.7, edgecolor='black')
axes[1].axvline(true_mu, color='red', linestyle='--', label=f'总体均值={true_mu}')
axes[1].set_xlabel('样本均值')
axes[1].set_ylabel('密度')
axes[1].set_title(f'样本均值的分布 (重复{n_samples}次抽样)')
axes[1].legend()
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('sample_distribution.png', dpi=100)
print("\n图像已保存: sample_distribution.png")
```

## 统计量的定义

### 统计量的概念

**定义**：设 $X_1, X_2, \ldots, X_n$ 是来自总体的样本，若 $T(X_1, \ldots, X_n)$ 是样本的函数，且**不依赖于任何未知参数**，则称 $T$ 为**统计量**。

### 统计量的特点

1. **是样本的函数**：由样本数据计算得到
2. **不含未知参数**：一旦样本确定，统计量的值就唯一确定
3. **是随机变量**：因为样本是随机的

### 统计量 vs 参数

| 概念 | 符号 | 含义 | 性质 |
|------|------|------|------|
| **参数** | $\theta$ | 描述总体特征的量 | 未知、固定 |
| **统计量** | $\hat{\theta}$ | 由样本计算的量 | 已知、随机 |

### 常见参数与对应的统计量

| 总体参数 | 符号 | 样本统计量 | 符号 |
|----------|------|------------|------|
| 总体均值 | $\mu$ | 样本均值 | $\bar{X}$ |
| 总体方差 | $\sigma^2$ | 样本方差 | $S^2$ |
| 总体比例 | $p$ | 样本比例 | $\hat{p}$ |
| 总体相关系数 | $\rho$ | 样本相关系数 | $R$ |

```python
import numpy as np

# 演示统计量
np.random.seed(42)
sample = np.random.normal(50, 10, 100)

# 定义各种统计量函数
def sample_mean(data):
    """样本均值"""
    return np.mean(data)

def sample_variance_unbiased(data):
    """无偏样本方差 (n-1)"""
    return np.var(data, ddof=1)

def sample_variance_biased(data):
    """有偏样本方差 (n)"""
    return np.var(data, ddof=0)

def sample_median(data):
    """样本中位数"""
    return np.median(data)

def sample_range(data):
    """样本极差"""
    return np.max(data) - np.min(data)

def sample_iqr(data):
    """四分位距"""
    q75, q25 = np.percentile(data, [75, 25])
    return q75 - q25

print("统计量计算示例:")
print("="*50)
print(f"样本容量: n = {len(sample)}")
print(f"样本均值: {sample_mean(sample):.4f}")
print(f"无偏样本方差: {sample_variance_unbiased(sample):.4f}")
print(f"有偏样本方差: {sample_variance_biased(sample):.4f}")
print(f"样本标准差: {np.sqrt(sample_variance_unbiased(sample)):.4f}")
print(f"样本中位数: {sample_median(sample):.4f}")
print(f"样本极差: {sample_range(sample):.4f}")
print(f"四分位距: {sample_iqr(sample):.4f}")
```

## 常用统计量

### 样本均值

$$
\bar{X} = \frac{1}{n} \sum_{i=1}^n X_i
$$

**性质**：
- **无偏性**：$\mathbb{E}[\bar{X}] = \mu$
- **方差**：$\text{Var}(\bar{X}) = \frac{\sigma^2}{n}$
- **一致性**：$\bar{X} \xrightarrow{P} \mu$（大数定律）

### 样本方差

$$
S^2 = \frac{1}{n-1} \sum_{i=1}^n (X_i - \bar{X})^2
$$

**为什么用 $n-1$？**

因为 $\bar{X}$ 是用样本估计的，会损失一个自由度。使用 $n-1$ 可以保证 $S^2$ 是 $\sigma^2$ 的**无偏估计**。

**样本方差无偏性的详细证明**：

**第一步**：展开平方和。

$$\sum_{i=1}^n (X_i - \bar{X})^2 = \sum_{i=1}^n (X_i - \mu + \mu - \bar{X})^2$$

$$= \sum_{i=1}^n [(X_i - \mu) - (\bar{X} - \mu)]^2$$

$$= \sum_{i=1}^n (X_i - \mu)^2 - 2(\bar{X} - \mu)\sum_{i=1}^n (X_i - \mu) + n(\bar{X} - \mu)^2$$

**第二步**：简化。

由于 $\sum_{i=1}^n (X_i - \mu) = n(\bar{X} - \mu)$，因此：

$$= \sum_{i=1}^n (X_i - \mu)^2 - n(\bar{X} - \mu)^2$$

**第三步**：计算期望。

$$\mathbb{E}\left[\sum_{i=1}^n (X_i - \mu)^2\right] = n\sigma^2$$

$$\mathbb{E}[(\bar{X} - \mu)^2] = \text{Var}(\bar{X}) = \frac{\sigma^2}{n}$$

因此：

$$\mathbb{E}\left[\sum_{i=1}^n (X_i - \bar{X})^2\right] = n\sigma^2 - n \cdot \frac{\sigma^2}{n} = (n-1)\sigma^2$$

**第四步**：得出结论。

$$\mathbb{E}[S^2] = \mathbb{E}\left[\frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2\right] = \frac{1}{n-1} \cdot (n-1)\sigma^2 = \sigma^2$$

$$\boxed{\mathbb{E}[S^2] = \sigma^2} \quad \text{（无偏估计）}$$

$$
\mathbb{E}[S^2] = \sigma^2
$$

**计算公式（简化形式）**：

$$
S^2 = \frac{1}{n-1}\left(\sum_{i=1}^n X_i^2 - n\bar{X}^2\right)
$$

### 样本标准差

$$
S = \sqrt{S^2}
$$

**注意**：$S$ 是 $\sigma$ 的**有偏估计**（Jensen 不等式）。

### 样本 k 阶矩

**k 阶原点矩**：

$$
A_k = \frac{1}{n} \sum_{i=1}^n X_i^k
$$

**k 阶中心矩**：

$$
B_k = \frac{1}{n} \sum_{i=1}^n (X_i - \bar{X})^k
$$

### 样本协方差（二维样本）

对于二维样本 $(X_1, Y_1), \ldots, (X_n, Y_n)$：

$$
S_{XY} = \frac{1}{n-1} \sum_{i=1}^n (X_i - \bar{X})(Y_i - \bar{Y})
$$

### 样本相关系数

$$
R = \frac{S_{XY}}{S_X S_Y} = \frac{\sum_{i=1}^n (X_i - \bar{X})(Y_i - \bar{Y})}{\sqrt{\sum_{i=1}^n (X_i - \bar{X})^2 \cdot \sum_{i=1}^n (Y_i - \bar{Y})^2}}
$$

```python
import numpy as np

# 深入理解样本方差的无偏性
np.random.seed(42)

true_mu = 10
true_sigma2 = 4
n_simulations = 10000
sample_sizes = [5, 10, 30, 100]

print("验证样本方差的无偏性:")
print("="*60)
print(f"真实方差: σ² = {true_sigma2}")
print()

for n in sample_sizes:
    biased_vars = []
    unbiased_vars = []
    
    for _ in range(n_simulations):
        sample = np.random.normal(true_mu, np.sqrt(true_sigma2), n)
        biased_vars.append(np.var(sample, ddof=0))  # 分母 n
        unbiased_vars.append(np.var(sample, ddof=1))  # 分母 n-1
    
    print(f"样本容量 n = {n}:")
    print(f"  有偏方差 E[S²_b]: {np.mean(biased_vars):.4f} (偏差: {np.mean(biased_vars) - true_sigma2:.4f})")
    print(f"  无偏方差 E[S²_u]: {np.mean(unbiased_vars):.4f} (偏差: {np.mean(unbiased_vars) - true_sigma2:.4f})")
    print()

# 样本协方差和相关系数示例
n = 100
x = np.random.randn(n)
y = 0.8 * x + 0.2 * np.random.randn(n)  # 相关数据

# 计算样本协方差和相关系数
sample_cov = np.cov(x, y)[0, 1]
sample_corr = np.corrcoef(x, y)[0, 1]

print("样本协方差和相关系数:")
print(f"  样本协方差: {sample_cov:.4f}")
print(f"  样本相关系数: {sample_corr:.4f}")
```

## 抽样分布

### 定义

**抽样分布**：统计量的概率分布。

由于统计量是随机变量，它有自己的分布。理解抽样分布是进行统计推断的基础。

### 样本均值的分布

**定理**：设 $X_1, \ldots, X_n \stackrel{i.i.d.}{\sim} \mathcal{N}(\mu, \sigma^2)$，则：

$$
\bar{X} \sim \mathcal{N}\left(\mu, \frac{\sigma^2}{n}\right)
$$

**标准化**：

$$
Z = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}} \sim \mathcal{N}(0, 1)
$$

**σ 未知时**：

$$
T = \frac{\bar{X} - \mu}{S / \sqrt{n}} \sim t(n-1)
$$

### 样本方差的分布

**定理**：设 $X_1, \ldots, X_n \stackrel{i.i.d.}{\sim} \mathcal{N}(\mu, \sigma^2)$，则：

$$
\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)
$$

**推论**：$\bar{X}$ 和 $S^2$ 相互独立。

### 样本均值与样本方差的独立性

对于正态总体，$\bar{X}$ 和 $S^2$ 是**相互独立**的。这是正态分布的重要性质，也是 t 分布推导的基础。

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 验证样本均值的分布
np.random.seed(42)
true_mu = 50
true_sigma = 10
n = 30
n_simulations = 10000

# 重复抽样计算样本均值
sample_means = []
sample_vars = []
for _ in range(n_simulations):
    sample = np.random.normal(true_mu, true_sigma, n)
    sample_means.append(sample.mean())
    sample_vars.append(sample.var(ddof=1))

sample_means = np.array(sample_means)
sample_vars = np.array(sample_vars)

# 理论分布参数
theoretical_mean = true_mu
theoretical_std = true_sigma / np.sqrt(n)

print("样本均值分布验证:")
print("="*50)
print(f"理论均值: {theoretical_mean:.4f}")
print(f"模拟均值: {sample_means.mean():.4f}")
print(f"理论标准差: {theoretical_std:.4f}")
print(f"模拟标准差: {sample_means.std():.4f}")

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(12, 4))

# 样本均值分布
axes[0].hist(sample_means, bins=50, density=True, alpha=0.7, edgecolor='black')
x = np.linspace(theoretical_mean - 4*theoretical_std, theoretical_mean + 4*theoretical_std, 100)
axes[0].plot(x, stats.norm.pdf(x, theoretical_mean, theoretical_std), 'r-', linewidth=2, label='理论分布')
axes[0].set_xlabel('样本均值')
axes[0].set_ylabel('密度')
axes[0].set_title(f'样本均值的分布 (n={n})')
axes[0].legend()
axes[0].grid(True, alpha=0.3)

# 标准化后的分布
standardized = (sample_means - true_mu) / (true_sigma / np.sqrt(n))
axes[1].hist(standardized, bins=50, density=True, alpha=0.7, edgecolor='black')
x = np.linspace(-4, 4, 100)
axes[1].plot(x, stats.norm.pdf(x, 0, 1), 'r-', linewidth=2, label='N(0,1)')
axes[1].set_xlabel('标准化样本均值')
axes[1].set_ylabel('密度')
axes[1].set_title('标准化后的分布')
axes[1].legend()
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('sampling_distribution_mean.png', dpi=100)
print("\n图像已保存: sampling_distribution_mean.png")
```

## 三大抽样分布

### χ² 分布（卡方分布）

#### 定义

若 $Z_1, Z_2, \ldots, Z_n \stackrel{i.i.d.}{\sim} \mathcal{N}(0, 1)$，则：

$$
\chi^2 = \sum_{i=1}^n Z_i^2 \sim \chi^2(n)
$$

其中 $n$ 称为**自由度**。

#### 性质

1. **期望**：$\mathbb{E}[\chi^2(n)] = n$
2. **方差**：$\text{Var}(\chi^2(n)) = 2n$
3. **可加性**：若 $\chi^2_1 \sim \chi^2(n_1)$，$\chi^2_2 \sim \chi^2(n_2)$，且独立，则 $\chi^2_1 + \chi^2_2 \sim \chi^2(n_1 + n_2)$
4. **渐近正态**：当 $n$ 很大时，$\chi^2(n) \approx \mathcal{N}(n, 2n)$

#### PDF

$$
f(x) = \frac{1}{2^{n/2}\Gamma(n/2)} x^{n/2-1} e^{-x/2}, \quad x > 0
$$

### t 分布

#### 定义

若 $Z \sim \mathcal{N}(0, 1)$，$V \sim \chi^2(n)$，且 $Z$ 与 $V$ 独立，则：

$$
T = \frac{Z}{\sqrt{V/n}} \sim t(n)
$$

#### 性质

1. **对称性**：关于 0 对称
2. **期望**：$\mathbb{E}[T] = 0$（当 $n > 1$）
3. **方差**：$\text{Var}(T) = \frac{n}{n-2}$（当 $n > 2$）
4. **尾部更厚**：比正态分布有更厚的尾部
5. **渐近正态**：当 $n \to \infty$ 时，$t(n) \to \mathcal{N}(0, 1)$

#### PDF

$$
f(x) = \frac{\Gamma\left(\frac{n+1}{2}\right)}{\sqrt{n\pi}\Gamma\left(\frac{n}{2}\right)} \left(1 + \frac{x^2}{n}\right)^{-\frac{n+1}{2}}
$$

### F 分布

#### 定义

若 $U \sim \chi^2(m)$，$V \sim \chi^2(n)$，且 $U$ 与 $V$ 独立，则：

$$
F = \frac{U/m}{V/n} \sim F(m, n)
$$

#### 性质

1. **期望**：$\mathbb{E}[F] = \frac{n}{n-2}$（当 $n > 2$）
2. **倒数关系**：若 $F \sim F(m, n)$，则 $1/F \sim F(n, m)$
3. **与 t 分布的关系**：$t^2(n) \sim F(1, n)$

### 三大分布的关系图

```
标准正态分布 Z ~ N(0,1)
        │
        ├── Z² → χ²(1)
        │
        ├── Z / √(χ²(n)/n) → t(n)
        │
        └── χ²(m)/m / (χ²(n)/n) → F(m,n)
```

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 可视化三大抽样分布
fig, axes = plt.subplots(1, 3, figsize=(15, 4))

# 1. χ² 分布
x = np.linspace(0, 30, 200)
for df in [1, 3, 5, 10, 20]:
    axes[0].plot(x, stats.chi2.pdf(x, df), label=f'ν={df}')
axes[0].set_xlabel('x')
axes[0].set_ylabel('f(x)')
axes[0].set_title('χ² 分布')
axes[0].legend()
axes[0].grid(True, alpha=0.3)
axes[0].set_ylim(0, 0.5)

# 2. t 分布
x = np.linspace(-5, 5, 200)
for df in [1, 3, 10, 30]:
    axes[1].plot(x, stats.t.pdf(x, df), label=f'ν={df}')
axes[1].plot(x, stats.norm.pdf(x), 'k--', label='N(0,1)')
axes[1].set_xlabel('x')
axes[1].set_ylabel('f(x)')
axes[1].set_title('t 分布（与标准正态对比）')
axes[1].legend()
axes[1].grid(True, alpha=0.3)

# 3. F 分布
x = np.linspace(0, 5, 200)
for m, n in [(1, 10), (5, 10), (10, 10), (10, 5)]:
    axes[2].plot(x, stats.f.pdf(x, m, n), label=f'({m},{n})')
axes[2].set_xlabel('x')
axes[2].set_ylabel('f(x)')
axes[2].set_title('F 分布')
axes[2].legend()
axes[2].grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('three_distributions.png', dpi=100)
print("图像已保存: three_distributions.png")

# 验证 t 分布的性质
print("\nt 分布性质验证:")
print("="*50)
n_simulations = 100000
df = 10

# 生成 t 分布随机数
z = np.random.randn(n_simulations)
v = np.random.chisquare(df, n_simulations)
t_samples = z / np.sqrt(v / df)

print(f"自由度 ν = {df}")
print(f"理论均值: 0, 模拟均值: {t_samples.mean():.4f}")
print(f"理论方差: {df/(df-2):.4f}, 模拟方差: {t_samples.var():.4f}")

# 与正态分布比较尾部
print(f"\n尾部概率比较 P(|T| > 2):")
print(f"t({df}) 分布: {2 * (1 - stats.t.cdf(2, df)):.4f}")
print(f"标准正态分布: {2 * (1 - stats.norm.cdf(2)):.4f}")
```

## 中心极限定理的应用

### 中心极限定理回顾

设 $X_1, \ldots, X_n \stackrel{i.i.d.}{\sim} F$，$\mathbb{E}[X_i] = \mu$，$\text{Var}(X_i) = \sigma^2$，则：

$$
\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} \mathcal{N}(0, 1)
$$

### 实际应用

**近似**：当 $n$ 足够大时（通常 $n \geq 30$）：

$$
\bar{X} \approx \mathcal{N}\left(\mu, \frac{\sigma^2}{n}\right)
$$

### 任意总体的情况

无论总体分布是什么（只要方差有限），样本均值都近似服从正态分布。

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# 中心极限定理演示：不同总体分布
np.random.seed(42)
n_simulations = 10000

# 定义不同的总体分布
distributions = {
    '均匀分布': lambda n: np.random.uniform(0, 1, n),
    '指数分布': lambda n: np.random.exponential(1, n),
    '伯努利分布': lambda n: np.random.binomial(1, 0.3, n),
}

sample_sizes = [5, 10, 30, 100]

fig, axes = plt.subplots(len(distributions), len(sample_sizes), figsize=(16, 10))

for i, (dist_name, dist_func) in enumerate(distributions.items()):
    for j, n in enumerate(sample_sizes):
        # 重复抽样计算样本均值
        sample_means = [dist_func(n).mean() for _ in range(n_simulations)]
        
        # 绘制直方图
        axes[i, j].hist(sample_means, bins=50, density=True, alpha=0.7, edgecolor='black')
        
        # 叠加正态分布（CLT 近似）
        mu = np.mean(sample_means)
        sigma = np.std(sample_means)
        x = np.linspace(mu - 4*sigma, mu + 4*sigma, 100)
        axes[i, j].plot(x, stats.norm.pdf(x, mu, sigma), 'r-', linewidth=2)
        
        axes[i, j].set_title(f'{dist_name}, n={n}')
        axes[i, j].grid(True, alpha=0.3)

plt.suptitle('中心极限定理：样本均值趋于正态分布', fontsize=14)
plt.tight_layout()
plt.savefig('clt_demonstration.png', dpi=100)
print("图像已保存: clt_demonstration.png")
```

## 在深度学习中的应用

### 1. Batch Normalization 的理论基础

Batch Normalization 使用**样本统计量**估计**总体统计量**：

$$
\hat{\mu}_B = \frac{1}{m}\sum_{i=1}^m x_i
$$

$$
\hat{\sigma}_B^2 = \frac{1}{m}\sum_{i=1}^m (x_i - \hat{\mu}_B)^2
$$

**大数定律保证**：当 batch size $m$ 足够大时，$\hat{\mu}_B \approx \mu$，$\hat{\sigma}_B^2 \approx \sigma^2$。

```python
import numpy as np

# Batch Normalization 统计量估计的准确性
np.random.seed(42)

true_mu = 0
true_sigma2 = 1
batch_sizes = [2, 8, 32, 128, 512]
n_simulations = 10000

print("Batch Normalization: 批统计量估计总体统计量")
print("="*60)
print(f"真实参数: μ = {true_mu}, σ² = {true_sigma2}")
print()

for m in batch_sizes:
    mu_estimates = []
    var_estimates = []
    
    for _ in range(n_simulations):
        batch = np.random.normal(true_mu, np.sqrt(true_sigma2), m)
        mu_estimates.append(batch.mean())
        var_estimates.append(batch.var(ddof=0))
    
    mu_estimates = np.array(mu_estimates)
    var_estimates = np.array(var_estimates)
    
    print(f"Batch size m = {m}:")
    print(f"  均值估计: E[μ̂] = {mu_estimates.mean():.4f}, Var(μ̂) = {mu_estimates.var():.4f}")
    print(f"  方差估计: E[σ̂²] = {var_estimates.mean():.4f}, Var(σ̂²) = {var_estimates.var():.4f}")
    print(f"  理论 Var(μ̂) = σ²/m = {true_sigma2/m:.4f}")
    print()
```

### 2. 模型评估的统计视角

**泛化误差**：

$$
R(\theta) = \mathbb{E}_{(x,y) \sim P_{data}}[\ell(f(x; \theta), y)]
$$

**经验风险**（训练误差）：

$$
\hat{R}(\theta) = \frac{1}{n}\sum_{i=1}^n \ell(f(x_i; \theta), y_i)
$$

**大数定律**：$\hat{R}(\theta) \xrightarrow{P} R(\theta)$

### 3. 梯度的随机估计

**真实梯度**：

$$
\nabla f(\theta) = \mathbb{E}_{(x,y)}[\nabla \ell(\theta; x, y)]
$$

**随机梯度估计**：

$$
\hat{g} = \frac{1}{B}\sum_{i \in \mathcal{B}} \nabla \ell(\theta; x_i, y_i)
$$

**性质**：
- $\mathbb{E}[\hat{g}] = \nabla f(\theta)$（无偏）
- $\text{Var}(\hat{g}) = O(1/B)$

### 4. 置信区间在模型比较中的应用

比较两个模型的性能差异时，可以使用**配对 t 检验**或**置信区间**：

```python
import numpy as np
from scipy import stats

# 模型比较示例
np.random.seed(42)

# 假设两个模型在 K 折交叉验证上的准确率
K = 10
model_a_acc = np.array([0.85, 0.87, 0.84, 0.86, 0.88, 0.85, 0.87, 0.86, 0.85, 0.87])
model_b_acc = np.array([0.82, 0.84, 0.81, 0.83, 0.85, 0.82, 0.84, 0.83, 0.82, 0.84])

# 配对差异
diff = model_a_acc - model_b_acc

# 计算置信区间
mean_diff = diff.mean()
se_diff = diff.std(ddof=1) / np.sqrt(K)
ci_95 = stats.t.interval(0.95, K-1, loc=mean_diff, scale=se_diff)

print("模型比较（配对差异）:")
print("="*50)
print(f"模型 A 平均准确率: {model_a_acc.mean():.4f}")
print(f"模型 B 平均准确率: {model_b_acc.mean():.4f}")
print(f"平均差异: {mean_diff:.4f}")
print(f"95% 置信区间: ({ci_95[0]:.4f}, {ci_95[1]:.4f})")

# t 检验
t_stat, p_value = stats.ttest_rel(model_a_acc, model_b_acc)
print(f"\nt 统计量: {t_stat:.4f}")
print(f"p 值: {p_value:.6f}")
print(f"结论: {'差异显著' if p_value < 0.05 else '差异不显著'} (α=0.05)")
```

## 小结

本章介绍了统计量与抽样分布的基础知识，这些是理解深度学习中统计估计的理论基础。

### 核心概念对照表

| 概念 | 公式/定义 | 深度学习应用 |
|------|----------|--------------|
| 样本均值 | $\bar{X} = \frac{1}{n}\sum X_i$ | Batch Normalization |
| 样本方差 | $S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$ | Batch Normalization |
| χ² 分布 | $\sum Z_i^2 \sim \chi^2(n)$ | 方差检验 |
| t 分布 | $Z/\sqrt{V/n} \sim t(n)$ | 置信区间、假设检验 |
| F 分布 | $\frac{U/m}{V/n} \sim F(m,n)$ | 方差比较 |

### 关键要点

1. **统计量**：从样本计算的量，是随机变量
2. **无偏性**：$\mathbb{E}[\hat{\theta}] = \theta$
3. **自由度**：样本方差用 $n-1$ 保证无偏
4. **抽样分布**：统计量的概率分布
5. **中心极限定理**：样本均值趋于正态

### 深度学习中的核心应用

| 技术 | 使用的统计概念 |
|------|---------------|
| Batch Normalization | 样本均值、样本方差 |
| SGD | 随机梯度估计、无偏性 |
| 模型评估 | 置信区间、假设检验 |
| 交叉验证 | 统计量稳定性 |
