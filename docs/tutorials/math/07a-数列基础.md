## 数列的定义与表示

### 基本定义

**数列**是按照一定顺序排列的一列数：

$$
\{a_n\} = a_1, a_2, a_3, \ldots, a_n, \ldots
$$

其中 $a_n$ 称为数列的**通项**（第 $n$ 项）。

### 表示方法

| 表示方法 | 示例 | 特点 |
|----------|------|------|
| 列举法 | $1, 3, 5, 7, 9, \ldots$ | 直观但冗长 |
| 通项公式 | $a_n = 2n - 1$ | 简洁明确 |
| 递推公式 | $a_1 = 1, a_{n+1} = a_n + 2$ | 适合编程实现 |
| 图像法 | 数轴上的点列 | 可视化 |

### 数列的分类

- **有穷数列**：项数有限，如 $1, 2, 3, 4, 5$
- **无穷数列**：项数无限，如 $1, \frac{1}{2}, \frac{1}{3}, \ldots$
- **有界数列**：存在 $M > 0$ 使得 $|a_n| \leq M, \forall n$
- **无界数列**：不存在这样的 $M$

```python
import numpy as np
import matplotlib.pyplot as plt

def visualize_sequence(sequence, title="数列可视化"):
    """可视化数列"""
    n = len(sequence)
    plt.figure(figsize=(10, 4))
    
    plt.subplot(1, 2, 1)
    plt.stem(range(1, n+1), sequence, basefmt=' ')
    plt.xlabel('n')
    plt.ylabel('$a_n$')
    plt.title(f'{title} - 散点图')
    plt.grid(True, alpha=0.3)
    
    plt.subplot(1, 2, 2)
    plt.plot(range(1, n+1), sequence, 'o-', markersize=4)
    plt.xlabel('n')
    plt.ylabel('$a_n$')
    plt.title(f'{title} - 折线图')
    plt.grid(True, alpha=0.3)
    
    plt.tight_layout()
    plt.show()

# 示例：递增数列
seq = [1/n for n in range(1, 21)]
visualize_sequence(seq, "递减数列 $a_n = 1/n$")
```

## 等差数列

### 定义

**等差数列**是相邻两项之差为常数的数列：

$$
a_{n+1} - a_n = d \quad (\text{常数})
$$

其中 $d$ 称为**公差**。

### 通项公式

$$
a_n = a_1 + (n-1)d
$$

**推导**：
$$
\begin{align}
a_2 &= a_1 + d \\
a_3 &= a_2 + d = a_1 + 2d \\
&\vdots \\
a_n &= a_1 + (n-1)d
\end{align}
$$

### 前 $n$ 项和公式

$$
S_n = \frac{n(a_1 + a_n)}{2} = \frac{n[2a_1 + (n-1)d]}{2}
$$

**推导**（高斯求和法）：

设 $S_n = a_1 + a_2 + \cdots + a_n$

则 $S_n = a_n + a_{n-1} + \cdots + a_1$（倒序）

两式相加：$2S_n = n(a_1 + a_n)$，故 $S_n = \frac{n(a_1 + a_n)}{2}$

### 性质

1. **等距性质**：若 $m + n = p + q$，则 $a_m + a_n = a_p + a_q$
2. **中项性质**：$2a_n = a_{n-1} + a_{n+1}$
3. **分段和**：$S_n, S_{2n} - S_n, S_{3n} - S_{2n}$ 仍成等差数列

```python
class ArithmeticSequence:
    """等差数列类"""
    def __init__(self, a1, d):
        self.a1 = a1  # 首项
        self.d = d    # 公差
    
    def term(self, n):
        """第n项"""
        return self.a1 + (n - 1) * self.d
    
    def sum(self, n):
        """前n项和"""
        return n * (self.a1 + self.term(n)) / 2
    
    def generate(self, n):
        """生成前n项"""
        return [self.term(i) for i in range(1, n+1)]

# 示例
seq = ArithmeticSequence(a1=1, d=2)
print(f"前10项: {seq.generate(10)}")
print(f"第10项: {seq.term(10)}")
print(f"前10项和: {seq.sum(10)}")

# 验证等距性质
print(f"a_2 + a_8 = {seq.term(2) + seq.term(8)}")
print(f"a_4 + a_6 = {seq.term(4) + seq.term(6)}")
print(f"a_3 + a_7 = {seq.term(3) + seq.term(7)}")  # 都等于 20
```

### 深度学习中的应用：线性学习率衰减

```python
def linear_lr_decay(initial_lr, final_lr, total_steps, current_step):
    """
    线性学习率衰减（等差数列形式）
    
    Args:
        initial_lr: 初始学习率
        final_lr: 最终学习率
        total_steps: 总步数
        current_step: 当前步数
    """
    d = (final_lr - initial_lr) / total_steps
    return initial_lr + current_step * d

# 可视化
import matplotlib.pyplot as plt

steps = 1000
lrs = [linear_lr_decay(0.1, 0.001, steps, s) for s in range(steps)]

plt.figure(figsize=(8, 4))
plt.plot(lrs)
plt.xlabel('Step')
plt.ylabel('Learning Rate')
plt.title('线性学习率衰减（等差数列）')
plt.grid(True, alpha=0.3)
plt.show()
```

## 等比数列

### 定义

**等比数列**是相邻两项之比为常数的数列：

$$
\frac{a_{n+1}}{a_n} = r \quad (\text{常数})
$$

其中 $r$ 称为**公比**。

### 通项公式

$$
a_n = a_1 \cdot r^{n-1}
$$

### 前 $n$ 项和公式

$$
S_n = \begin{cases}
na_1 & r = 1 \\
a_1 \frac{1 - r^n}{1 - r} & r \neq 1
\end{cases}
$$

**推导**（错位相减法）：

设 $S_n = a_1 + a_1 r + a_1 r^2 + \cdots + a_1 r^{n-1}$

则 $rS_n = a_1 r + a_1 r^2 + \cdots + a_1 r^{n-1} + a_1 r^n$

相减：$(1-r)S_n = a_1 - a_1 r^n = a_1(1 - r^n)$

故 $S_n = a_1 \frac{1 - r^n}{1 - r}$

### 无穷等比级数

当 $|r| < 1$ 时，无穷等比级数收敛：

$$
\sum_{n=0}^{\infty} a_1 r^n = \frac{a_1}{1 - r}
$$

### 性质

1. **等距性质**：若 $m + n = p + q$，则 $a_m \cdot a_n = a_p \cdot a_q$
2. **中项性质**：$a_n^2 = a_{n-1} \cdot a_{n+1}$
3. **分段积**：相邻 $k$ 项的积构成新的等比数列

```python
class GeometricSequence:
    """等比数列类"""
    def __init__(self, a1, r):
        self.a1 = a1  # 首项
        self.r = r    # 公比
    
    def term(self, n):
        """第n项"""
        return self.a1 * (self.r ** (n - 1))
    
    def sum(self, n):
        """前n项和"""
        if abs(self.r - 1) < 1e-10:
            return n * self.a1
        return self.a1 * (1 - self.r**n) / (1 - self.r)
    
    def infinite_sum(self):
        """无穷级数和（仅当|r| < 1时收敛）"""
        if abs(self.r) >= 1:
            return float('inf')  # 发散
        return self.a1 / (1 - self.r)
    
    def generate(self, n):
        """生成前n项"""
        return [self.term(i) for i in range(1, n+1)]

# 示例
seq = GeometricSequence(a1=1, r=0.5)
print(f"前10项: {seq.generate(10)}")
print(f"第10项: {seq.term(10)}")
print(f"前10项和: {seq.sum(10):.4f}")
print(f"无穷级数和: {seq.infinite_sum():.4f}")  # 理论值 = 2

# 验证中项性质
print(f"a_5² = {seq.term(5)**2:.6f}")
print(f"a_4 × a_6 = {seq.term(4) * seq.term(6):.6f}")
```

### 深度学习中的应用：指数学习率衰减

```python
def exponential_lr_decay(initial_lr, decay_rate, decay_steps, current_step):
    """
    指数学习率衰减（等比数列形式）
    
    Args:
        initial_lr: 初始学习率
        decay_rate: 衰减率（公比）
        decay_steps: 衰减周期
        current_step: 当前步数
    """
    return initial_lr * (decay_rate ** (current_step / decay_steps))

# 不同衰减率的比较
import matplotlib.pyplot as plt

plt.figure(figsize=(10, 5))
for rate in decay_rates:
    lrs = [exponential_lr_decay(0.1, rate, 100, s) for s in range(steps)]
    plt.plot(lrs, label=f'decay_rate={rate}')

plt.xlabel('Step')
plt.ylabel('Learning Rate')
plt.title('指数学习率衰减（等比数列）')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

## 递推数列

### 定义

**递推数列**是通过递推关系定义的数列，每一项由前几项通过某种规则得到。

### 一阶递推

$$
a_{n+1} = f(a_n)
$$

**例子**：$a_{n+1} = 2a_n + 1$，$a_1 = 1$

### 二阶递推

$$
a_{n+2} = f(a_{n+1}, a_n)
$$

**斐波那契数列**：$F_{n+2} = F_{n+1} + F_n$，$F_1 = F_2 = 1$

### 线性递推关系的一般解法

对于 $a_{n+2} = pa_{n+1} + qa_n$：

1. 写出**特征方程**：$x^2 - px - q = 0$
2. 求特征根 $\alpha, \beta$
3. 根据根的情况写出通解：
   - 若 $\alpha \neq \beta$：$a_n = A \cdot \alpha^n + B \cdot \beta^n$
   - 若 $\alpha = \beta$：$a_n = (A + Bn) \cdot \alpha^n$
4. 用初始条件确定 $A, B$

```python
def fibonacci(n):
    """斐波那契数列"""
    if n <= 2:
        return 1
    return fibonacci(n-1) + fibonacci(n-2)

def fibonacci_iterative(n):
    """斐波那契数列（迭代版本，更高效）"""
    if n <= 2:
        return 1
    a, b = 1, 1
    for _ in range(n - 2):
        a, b = b, a + b
    return b

# 使用通项公式
def fibonacci_closed_form(n):
    """斐波那契数列通项公式"""
    phi = (1 + np.sqrt(5)) / 2  # 黄金比例
    psi = (1 - np.sqrt(5)) / 2
    return int((phi**n - psi**n) / np.sqrt(5))

# 验证
print("前20项斐波那契数列:")
fib_seq = [fibonacci_iterative(i) for i in range(1, 21)]
print(fib_seq)

# 验证通项公式
print("\n验证通项公式:")
for i in range(1, 11):
    print(f"F_{i}: 迭代={fibonacci_iterative(i)}, 公式={fibonacci_closed_form(i)}")
```

### 深度学习中的应用：RNN 隐状态

RNN 的隐状态更新就是一个递推关系：

$$
h_t = \tanh(W_h h_{t-1} + W_x x_t + b)
$$

```python
class SimpleRNN:
    """简单 RNN 演示（递推数列形式）"""
    def __init__(self, hidden_size, input_size):
        self.W_h = np.random.randn(hidden_size, hidden_size) * 0.01
        self.W_x = np.random.randn(hidden_size, input_size) * 0.01
        self.b = np.zeros(hidden_size)
        self.hidden_size = hidden_size
    
    def forward(self, x_sequence):
        """
        前向传播 - 递推计算隐状态序列
        
        Args:
            x_sequence: 输入序列 (seq_len, input_size)
        """
        seq_len = x_sequence.shape[0]
        h_sequence = np.zeros((seq_len + 1, self.hidden_size))
        
        for t in range(seq_len):
            # 递推关系: h_t = f(h_{t-1}, x_t)
            h_sequence[t+1] = np.tanh(
                self.W_h @ h_sequence[t] + 
                self.W_x @ x_sequence[t] + 
                self.b
            )
        
        return h_sequence[1:]  # 返回所有时刻的隐状态

# 示例
rnn = SimpleRNN(hidden_size=4, input_size=2)
x = np.random.randn(5, 2)  # 序列长度为5
h = rnn.forward(x)
print("RNN 隐状态序列（递推数列）:")
print(h)
```

## 数列的性质

### 单调性

- **单调递增**：$a_{n+1} > a_n, \forall n$
- **单调递减**：$a_{n+1} < a_n, \forall n$
- **单调不减**：$a_{n+1} \geq a_n, \forall n$
- **单调不增**：$a_{n+1} \leq a_n, \forall n$

### 有界性

- **上界**：$\exists M, a_n \leq M, \forall n$
- **下界**：$\exists m, a_n \geq m, \forall n$
- **有界**：既有上界又有下界

### 单调有界定理

**单调有界数列必收敛**。

这是判断数列极限存在的重要工具。

```python
def analyze_sequence(sequence, name):
    """分析数列的性质"""
    arr = np.array(sequence)
    
    # 单调性
    diffs = np.diff(arr)
    if np.all(diffs > 0):
        monotonicity = "单调递增"
    elif np.all(diffs < 0):
        monotonicity = "单调递减"
    elif np.all(diffs >= 0):
        monotonicity = "单调不减"
    elif np.all(diffs <= 0):
        monotonicity = "单调不增"
    else:
        monotonicity = "非单调"
    
    # 有界性
    if np.isfinite(arr).all():
        bounded = f"有界 [{arr.min():.4f}, {arr.max():.4f}]"
    else:
        bounded = "无界"
    
    print(f"{name}:")
    print(f"  单调性: {monotonicity}")
    print(f"  有界性: {bounded}")
    print(f"  前5项: {arr[:5]}")
    print(f"  后5项: {arr[-5:]}")
    print()

# 分析各种数列
analyze_sequence([1/n for n in range(1, 100)], "$a_n = 1/n$")
analyze_sequence([np.log(n) for n in range(1, 100)], "$a_n = \\ln n$")
analyze_sequence([1 - 1/n for n in range(1, 100)], "$a_n = 1 - 1/n$")
analyze_sequence([(-1)**n / n for n in range(1, 100)], "$a_n = (-1)^n / n$")
```

## 在深度学习中的应用

### 1. 学习率调度

| 策略 | 数列类型 | 公式 |
|------|----------|------|
| 线性衰减 | 等差数列 | $\eta_t = \eta_0 - kt$ |
| 指数衰减 | 等比数列 | $\eta_t = \eta_0 \cdot \gamma^t$ |
| 阶梯衰减 | 分段常数 | $\eta_t = \eta_0 \cdot \gamma^{\lfloor t/T \rfloor}$ |

### 2. RNN 梯度分析

RNN 梯度的核心是矩阵乘积的等比级数：

$$
\frac{\partial h_T}{\partial h_1} = \prod_{t=1}^{T-1} W_h
$$

当 $W_h$ 的最大特征值 $|\lambda_{\max}| < 1$ 时，梯度指数衰减（梯度消失）；当 $|\lambda_{\max}| > 1$ 时，梯度指数增长（梯度爆炸）。

### 3. 位置编码

Transformer 的正弦位置编码使用不同频率的等比数列：

$$
PE_{(pos, 2i)} = \sin\left(\frac{pos}{10000^{2i/d}}\right)
$$

其中 $10000^{2i/d}$ 构成等比数列。

```python
def positional_encoding(max_len, d_model):
    """Transformer 位置编码"""
    position = np.arange(max_len)[:, np.newaxis]
    div_term = np.exp(np.arange(0, d_model, 2) * (-np.log(10000.0) / d_model))
    
    pe = np.zeros((max_len, d_model))
    pe[:, 0::2] = np.sin(position * div_term)  # 偶数维度
    pe[:, 1::2] = np.cos(position * div_term)  # 奇数维度
    
    return pe

# 可视化位置编码
pe = positional_encoding(100, 64)

plt.figure(figsize=(12, 4))
plt.subplot(1, 2, 1)
plt.imshow(pe, aspect='auto', cmap='coolwarm')
plt.colorbar(label='Encoding Value')
plt.xlabel('Dimension')
plt.ylabel('Position')
plt.title('位置编码矩阵')

plt.subplot(1, 2, 2)
# 显示 div_term（等比数列）
div_term = np.exp(np.arange(0, 64, 2) * (-np.log(10000.0) / 64))
plt.semilogy(div_term, 'o-', markersize=4)
plt.xlabel('Dimension Index')
plt.ylabel('Frequency Divisor (log scale)')
plt.title('频率因子（等比数列）')
plt.grid(True)

plt.tight_layout()
plt.show()
```

## 小结

本节介绍了数列的基本概念：

| 概念 | 定义/公式 | 特点 |
|------|----------|------|
| 等差数列 | $a_n = a_1 + (n-1)d$ | 相邻项差为常数 |
| 等比数列 | $a_n = a_1 \cdot r^{n-1}$ | 相邻项比为常数 |
| 递推数列 | $a_n = f(a_{n-1}, \ldots)$ | 由前项推出后项 |
| 单调性 | $a_{n+1} \geq a_n$ 或 $a_{n+1} \leq a_n$ | 变化趋势 |
| 有界性 | $\|a_n\| \leq M$ | 值的范围 |

**关键要点**：
- 等差数列用于线性学习率衰减
- 等比数列用于指数衰减、RNN 梯度分析
- 递推数列是 RNN 的数学本质
- 单调有界数列必收敛
