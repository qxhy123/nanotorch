# 第三章：概率论

概率论是理解**机器学习不确定性**和**统计学习理论**的基础。从分类问题的 Softmax 输出到生成模型，从 Dropout 正则化到贝叶斯神经网络，概率论无处不在。本章将系统介绍深度学习所需的概率论知识。

## 为什么概率论对深度学习重要？

### 1. 数据的不确定性

```
真实世界数据 → 含噪声 → 概率模型
    ↓
测量误差、标注噪声、数据缺失 → 需要概率建模
```

### 2. 模型的概率解释

| 模型组件 | 概率解释 |
|----------|----------|
| Softmax 输出 | 类别分布 $P(y|x)$ |
| Sigmoid 输出 | 伯努利分布参数 $p$ |
| MSE 损失 | 高斯噪声假设下的负对数似然 |
| Cross Entropy | 最大似然估计 |
| Dropout | 贝叶斯近似的采样 |

### 3. 生成模型的核心

- **VAE**：学习数据的潜在分布
- **GAN**：生成分布逼近真实分布
- **Diffusion**：逐步去噪的概率过程

### 4. 不确定性量化

```python
# 例子：预测不确定性
prediction = model(x)  # 点估计
prediction_dist = bayesian_model(x)  # 分布估计
# 可以回答："模型有多大把握？"
```

## 核心公式速查

### 概率基础

$$
P(A|B) = \frac{P(A \cap B)}{P(B)}
$$

$$
P(B|A) = \frac{P(A|B)P(B)}{P(A)} \quad \text{(贝叶斯定理)}
$$

### 期望与方差

$$
\mathbb{E}[X] = \sum_x x \cdot p(x) \quad \text{(离散)}
$$

$$
\text{Var}(X) = \mathbb{E}[X^2] - (\mathbb{E}[X])^2
$$

### 正态分布

$$
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)
$$

### 交叉熵损失

$$
L = -\sum_{i=1}^K y_i \log \hat{y}_i
$$

### KL 散度

$$
D_{KL}(P \| Q) = \sum_x p(x) \log \frac{p(x)}{q(x)}
$$

## Python 代码示例

### Softmax 与交叉熵

```python
import numpy as np

def softmax(logits):
    """将 logits 转换为概率分布"""
    exp_logits = np.exp(logits - np.max(logits))
    return exp_logits / exp_logits.sum()

def cross_entropy_loss(probs, target):
    """交叉熵损失"""
    return -np.log(probs[target])

# 示例
logits = np.array([2.0, 1.0, 0.1])
probs = softmax(logits)
print(f"概率分布: {probs}")  # [0.659, 0.242, 0.099]
print(f"预测类别 0 的损失: {cross_entropy_loss(probs, 0):.4f}")
```

### 正态分布初始化

```python
import numpy as np

def he_init(fan_in, fan_out):
    """He 初始化（用于 ReLU）"""
    std = np.sqrt(2 / fan_in)
    return np.random.normal(0, std, (fan_out, fan_in))

weights = he_init(784, 256)
print(f"权重形状: {weights.shape}")
print(f"理论标准差: {np.sqrt(2/784):.4f}")
```
